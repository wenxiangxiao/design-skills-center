About · 關於平台

把資料還給市民。

青嶼市開放資料平台（CIVICA）由市政府資訊局於 2019 年建立，目標只有一個：讓任何人都能查、能下載、能重用市政資料，不必寫公文申請。這裡沒有登入牆，也沒有付費牆。

01三項原則

預設開放資料以公開為原則、不公開為例外。除涉個資與國安者外，一律上架。
機器可讀提供 CSV／JSON／GeoJSON 與 REST API，欄位有中英對照的資料字典。
自由重用採 CC0 公眾領域標示，商用、改作、再散布皆免授權費、免申請。

02API 使用說明

所有資料集皆可透過統一的 REST 端點取得。無需申請金鑰即可查詢，但匿名呼叫限每分鐘 60 次；註冊金鑰後放寬至每分鐘 600 次。

# 取得「公車即時到站」資料集前 100 筆
GET https://api.civica.qingyu.gov.example/v2/datasets/QY-TRN-007/records?limit=100

# 回應（節錄）
{ "dataset": "QY-TRN-007",
  "count": 100,
  "license": "CC0-1.0",
  "records": [ … ] }
端點用途格式
/v2/datasets列出全部資料集與中繼資料JSON
/v2/datasets/{id}/records取得單一資料集的紀錄JSON / CSV
/v2/datasets/{id}/schema取得欄位資料字典JSON
/v2/geo/{id}取得地理圖層GeoJSON
