## 首創登記

**確定性偽即時多人（pseudo-multiplayer）**作為互動範式與全站的活力來源——全館第一個「別的訪客看似此刻正在參與」的站。24 位聯盟球員由 `seed ⊕ gameIndex` 與真實時鐘確定性驅動：每人此刻打到第幾格、該格幾瓶、即時分是多少，全是 `f(seed, wall-clock time)` 的純函數，換誰看、重新整理幾次結果都一致、離線期間可重算而非暫停。使用者以相同的十瓶計分規則投出真實一局後注入同一榜單、與模擬球員同欄競名。與 beitun-bell（單一對手的即時對抗賽局）不同：本站是**多人共存的活榜**而非一對一賽局；與 tickerhouse／nightspin 的「報價帶捲動」不同：那是裝飾性跑馬燈，本站的即時榜是由每位球員的比賽狀態驅動、且使用者能真正進榜的功能性儀表板。
