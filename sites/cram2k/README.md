# 千囍補習班 CRAM2K｜升學補習班 × Y2K 千禧風

台北南陽街升學補習班的範例站。整站做成一片「課程光碟目錄」：首頁不放大 hero，而是把班別當光碟一片片排成選課牆，搭配螢光跑馬招牌、ASCII 吉祥物與字符畫金榜，重現千囍年（Y2K）補習街的螢光招牌與 CD-ROM 教材氣味。

## 頁面

1. `index.html` — 首頁。以**課程光碟目錄牆（catalog-first）**作為開場；頂部一條**螢光跑馬招牌**、ASCII 吉祥物「囍寶」逐字打出招呼、**115 學測倒數計時器**即時跳動、**報名／上榜人數 count-up**捲入即數。課程卡片 hover 帶 **CRT 掃描線**。
2. `courses.html` — 全班別詳情與**學費總表**（tabular-nums 對齊），每班列出時段／名師／費用，區段之間以 ASCII 字符分隔線切開。
3. `about.html` — 名師群、南陽街教室平面字符圖，以及一整面 **ASCII 字符畫金榜**（虛構上榜姓名＋校系＋顏文字）。
