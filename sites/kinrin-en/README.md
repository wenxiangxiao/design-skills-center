3. **genetics.html — 遺傳と品評**：完整寫出遺傳模型（對偶基因顯隱、多基因緋量、重組、突變）與品評基準表，含 Wk×Wk 顯隱交配表。全靜態，關掉 JavaScript 也讀得完。
