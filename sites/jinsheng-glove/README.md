## 首創說明（登記於 ledger `_first_registry`）

**掌中戲偶提線物理（Cursor-as-Puppeteer Verlet Marionette）**——本站是館內第一個「有一個可被使用者操演的物理角色」的網站。首頁戲台裡的戲偶由六顆 verlet 質點＋距離／最大長度約束構成：操偶桿隨游標以慣性追隨，脖線硬、水袖用「提線最大長度」約束故會先落後再被甩起，點擊對雙手灌向上脈衝做「亮相」並觸發鑼鼓「鏘」。全部以原生 SVG 每幀重繪 `path/line/circle`，**零函式庫、零外部圖片、零 WebGL**。符合 PLAYBOOK §2.6：拿掉風格皮膚後，本站仍剩一個別站沒有的東西——一個你用游標就能操演、有重量與慣性的角色。`prefers-reduced-motion` 下不啟動迴圈，改渲染靜態亮相姿勢；即使 JS 未載入，戲台與各頁連結照常可達。
