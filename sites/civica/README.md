# 青嶼市開放資料平台 CIVICA — 瑞士國際主義（Swiss International）

> Design Skills Center 館藏範例站 · 產業 × 風格：**政府開放資料 × 瑞士國際主義 Swiss International**

一個虛構城市「青嶼市」的開放資料平台。CIVICA 把首頁做成一份**可用的目錄索引**——不是搜尋框、不是大標，而是按主題編號排列的資料清單，點進去就能看圖、下載、接 API。整站示範 **瑞士國際主義（Swiss International / International Typographic Style）**：嚴謹網格、齊左的無襯線黑體、克制的紅黑白（此站改用墨黑＋國際藍）、以資訊層級而非裝飾說話。

政府／資料題材天生適合瑞士風的客觀與秩序感。為與館內既有的瑞士風美術館站（RASTER，紅色系）拉開，本站改用**墨黑＋國際藍（`#243FD6`）**的訊號色，並把重點放在原創 SVG 資料圖表。

## 版面與骨架（對照 PLAYBOOK §2.4）

- **開場原型 opening：`toc-first` 目錄索引開場**——首頁主體是一份六大主題的編號索引（財政、交通、環境、人口、設施、安全），每列含編號、標題、資料集數與說明，hover 整列反黑。這一頁本身就是導覽，不是裝飾性 hero。全館第一個 toc-first。
- **導覽原型 nav：`side-rail` 垂直側欄**——左側固定 232px 側欄，含 2px 方格 logo、編號式垂直選單、最後更新日期與「資料皆為虛構」聲明；手機收合為頂部橫列。非 topbar。
- **動效簽名 signature：資料圖表捲動進場逐欄升起＋游標十字準星讀值＋2px 網格基線常駐**——原創 SVG 長條圖以 `IntersectionObserver` 觸發、依序 `scaleY` 由基線長高；折線圖以 `stroke-dashoffset` 描出；滑鼠移入顯示十字準星與讀數。全館唯一。

## 頁面

1. `index.html` — 目錄索引首頁：刊頭統計、六大主題編號索引、三張本月市政指標圖表（公車運量長條、PM2.5 折線、預算執行率長條）。
2. `datasets.html` — 資料集清單：主題篩選晶片、12 筆資料集（格式／編號／更新頻率／下載數／API），並附「資料速覽」年齡結構長條圖與環圈占比圖。
3. `about.html` — 關於與 API：三項開放原則、REST API 說明與端點表、CC0 授權與引用、聯絡資訊。

## 品牌設定

- **平台**：青嶼市開放資料平台 CIVICA（虛構示範城市）
- **一句話**：市政資料，攤在桌上。
- **主辦**：青嶼市政府資訊局 資料治理科（虛構）
- **地址電話**：青嶼市中正路 1 號 6 樓 ／ 02-8900-1200 轉 3021 ／ opendata@qingyu.gov.example（皆為虛構）
- **規模示意**：248 個資料集、6 大主題、累計下載 31.2M、授權 CC0
- **代表色**：紙白 `#F0EEE9`／墨黑 `#111111`／國際藍 `#243FD6`／中性灰 `#8C887E`／淺分隔 `#DBD8D0`

## 去AI化重點

- 無紫藍漸層 hero；首頁是編號目錄索引，資訊層級即版面。
- 無「置中大標＋兩顆按鈕＋三張圓角卡」模板；用側欄＋齊左網格＋2px 實線分格。
- 所有圖表為原創 inline SVG（長條、折線、環圈），零外部圖片、零圖表函式庫；外部資源僅 Google Fonts。
- 無圓角、無模糊陰影；瑞士風靠網格、留白與字級對比。
- 無跑馬燈——資訊動效交給圖表進場與十字準星讀值。
- 敏感題材（政府資料）全站標示「青嶼市為虛構示範城市，資料皆為虛構示意」。
