# 捏程 KNEAD — 黏土擬物（Claymorphism）

> Design Skills Center 館藏範例站 · 產業 × 風格：**線上課程 × 黏土擬物 Claymorphism**

一個虛構的線上學習平台。捏程 KNEAD 不把課程當貨架讓你自己海選——你回答三個問題，平台替你「捏」出一條有先後、有節奏的學習路徑。整站示範 **黏土擬物（Claymorphism）** 這套視覺語言：柔軟的膨脹造型、雙向柔光陰影（左上高光＋右下暗影）塑造的浮起感、按下去會凹陷的觸感元件、溫潤的麵團暖色。

刻意避開紫藍冷色與冰冷科技感——用**麵團米 × 陶土棕 × 蜜桃橘 × 鼠尾草綠**的暖色系，讓黏土回到「手指沾到黏土那一下」的手作溫度。

## 版面與骨架（對照 PLAYBOOK §2.4）

- **開場原型 opening：`form-first` 對話/表單開場**——首頁不是大標 hero，而是一組三步驟互動測驗（領域 → 程度 → 每週時間）。使用者一進來就在動手回答，最後系統把答案揉成一條可視化的學習路徑。全館第一個 form-first。
- **導覽原型 nav：`bottom-dock` 底部懸浮塢**——導覽做成畫面底部懸浮的黏土塢，三顆原創 SVG 圖示（捏路徑／課程／關於），像 App 底欄。非 topbar。
- **動效簽名 signature：黏土按壓凹陷（雙向柔光內外陰影切換）＋捏塑學習路徑逐步成形**——所有可按元件靜置時凸起、按下／選中時內凹（`box-shadow` outset↔inset 互換），配合彈性回彈 easing；三步測驗完成後，學習路徑的黏土 blob 逐塊揉出、以陶土色連桿相接。全館唯一。

## 去AI化重點

- 無紫藍漸層 hero；首頁是可操作的表單測驗，不是置中大標。
- 無「置中大標＋兩顆按鈕＋三張圓角卡」模板；用不對稱的表單開場與底部塢導覽。
- 黏土風本就用柔和陰影，但這裡是**有意的風格語彙**（按壓觸感 = 核心互動），非隨手套用的模糊卡片；所有圖示為原創 SVG，零 emoji。
- 所有圖像為原創 SVG／CSS，零外部圖片；外部資源僅 Google Fonts。
- 無跑馬燈——黏土風靠按壓觸感與路徑成形承擔動效重點。
- 人物、價格、地址均標示為虛構示意。
