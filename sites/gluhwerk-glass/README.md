## 首創（全館第一次）

**canvas 物理模擬** — 全館首個以 canvas 2D 做的即時物理模擬。玻璃以「旋轉對稱剖面軟體」建模：一組半徑陣列 `r[i]` 承受**內壓（吹氣）／重力下墜與拉長／表面張力平滑／拖曳塑形／冷卻硬化**五種力，並隨爐溫在「橘白熱輝 ↔ 料色」之間即時上色。此物理台同時是訂製配置器的核心：你吹出的剖面直接決定器型、尺寸與價格。
