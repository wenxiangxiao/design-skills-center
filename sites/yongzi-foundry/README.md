**產業**：中文字體鑄造所（Type Foundry）　**風格**：字骨規格書 Skeleton-Spec

台南中西區一棟老鐵工廠二樓的字體工作室。主持人石文彬，做過十四年報社檢字與拼版；

## 這個站的核心：網站的字是自己鑄出來的

**全站所有中文字符——招牌、導覽鉛字、字樣盤、比對台、瀏覽器分頁圖示——都不是字型檔。**
每個字是一組筆畫骨架（`h` 橫／`v` 豎／`d` 撇捺／`p` 點／`z` 折，寫在 1000×1000 字身框裡），
經過八項參數即時合成的 SVG：

| 桿 | 參數 | 作用 |
|---|---|---|
| 字重 WEIGHT | `w` 26–190 u | 豎筆寬度 |
| 橫豎對比 CONTRAST | `c` 0–0.8 | 橫筆寬 = 豎筆寬 ×(1−c×0.78) |
| 字面率 WIDTH | `x` 0.78–1.10 | 骨架水平放縮 |
| 中宮 COUNTER | `g` 0.86–1.14 | 骨架對中心放縮，決定內白鬆緊 |
| 襯線量 SERIF | `s` 0–1 | 橫畫起筆逆入三角、收筆頓角、豎畫起筆左右突 |
| 傾斜 SLANT | `k` ±12° | `skewX` |
| 筆端 TERMINAL | `e` 0/1 | `stroke-linecap` 平切／圓收 |
| 轉角 JOIN | `j` 0/1 | `stroke-linejoin` 方角／圓角 |

站上六套「零售字體」（黑體／明體／圓體／超黑／細等／斜宋）不是六個字型檔，
是同一批字骨的六組參數。這正是本站要示範的事：**字體的差別可以被寫成數字，
但把中間值修到能賣，是人的工作**——授權頁的 FAQ 用鑄字師的口氣把這件事講清楚了。

## 首創（全館第一次）

**字形合成引擎作為網站自身的識別——使用者可以改鑄這個網站。**

全館 87 站的字體全部來自 Google Fonts；本站的中文字沒有字型檔，
而且這件事是**可被訪客接管**的：拖動鑄字桿按下「套用到全站」，
招牌、四枚導覽鉛字、瀏覽器分頁 favicon（即時重建的 data URI）與所有字樣同時改鑄，
狀態寫入 `sessionStorage` 跨頁生效，並可用鑄體碼 `?f=` 分享給別人看你調出來的版本。
訪客在鑄字檯補鑄的缺字也會真的進入字符盤，跟著現行鑄體一起變胖變斜。

與既有首創的區隔：`shaft-eight` 的織法引擎是全站唯一渲染器，但渲染的是紋樣；
本站渲染的是**文字**這個網站最基本的載體，且品牌識別本身被交到訪客手上。

## 無障礙與破格出口

- 導覽鉛字預設鏡像不可讀（活版原理），但每枚下方恆有正體中文標籤 + 英文，`aria-label` 為正字，行動版直接顯示正字，頁尾另備完整連結。
- 鑄字檯備「照範例補鑄」按鈕，無指標裝置或不想畫的人一鍵放上標準骨架；折筆有「收折」按鈕。
- `prefers-reduced-motion` 下所有過渡壓至 0.001ms；本站無自動播放動畫，無任何邏輯被停用。
- 授權頁與所有文案為靜態 HTML，關閉 JS 仍可完整閱讀；四頁皆附 `<noscript>` 替代管道。
- 零外部圖片、零音檔、零 3D、零第三方函式庫；外部資源僅 Google Fonts（Fraunces + Space Mono + Noto Sans TC，皆為拉丁與介面字，中文字符不依賴它們）。
