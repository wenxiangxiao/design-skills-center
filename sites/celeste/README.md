# CELESTE 藍晒天象館

**產業**：天文館　·　**風格**：藍晒 Cyanotype

一座把星圖當成顯影過程的虛構天象館。整站只有兩種顏色——普魯士藍與藍晒紙的米白——像一張攤開的十九世紀天體攝影負片，卻裝著一台 1968 年的機械投影機。

## 八維配方

> **天文館 × 藍晒 Cyanotype × 1960s（太空時代）× 無地域感 × 冷靜理性 × 紙感（藍晒印樣）× 中等密度 × 圓頂選位訂票**

- **era**：1960s——太空競賽年代的樂觀，混進維多利亞藍晒的古典。
- **locale**：無地域感——星圖是全人類共通的語言，品牌不綁土地。
- **voice**：冷靜理性——像天文台值班員的口吻，精確、克制、不煽情。
- **texture**：紙感——普魯士藍上的一點白，如曝曬過的鐵鹽印樣。
- **density**：中等——留白給天空，資訊集中在星盤與座位圖。
- **function（核心可玩功能）**：圓頂劇場**線上選位訂票**——挑場次 → 在環形座位圖點選座位 → 即時結算 → 取得訂位代碼。

## 頁面

- `index.html`　星圖首頁（map-first）：可拖曳旋轉的藍晒星盤（planisphere），旋轉外環即改變恆星時與正對子午線的星座讀值。
- `program.html`　節目與訂票：今日場次清單 + 環形座位圖選位、即時結算、訂位代碼。
- `visit.html`　參觀資訊：開館時間、票價、藍晒風路線圖、FAQ。

## 版面原型

- **opening**：map-first（星圖即首頁）
- **nav**：bottom-dock（天象儀控制台底欄）
