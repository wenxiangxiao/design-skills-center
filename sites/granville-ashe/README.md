## 開場與導覽原型

- **開場（opening）**：`masthead` — 首頁第一屏是對稱的刊頭銘牌，像一張被鏨刻的信箋抬頭，把事務所的名號與年份當作主角，而非行銷式大標與按鈕。
- **導覽（nav）**：`masthead` — 置中的刊頭式導覽，事務所名、副標與導覽列上下對齊置中，維持古典對稱感（非置頂列 topbar）。
- **動效簽名（signature）**：字標載入金箔光澤掃過（sheen sweep）＋執業領域羅馬數字索引 hover／點擊展開時金色髮線由左自繪並揭示代表案件標的、數字滾動計數＋放射狀日芒徽記極慢旋轉。全館其他站沒有「羅馬數字執業索引 × 金箔掃光 × 日芒緩轉」這一組合。
