# 鏽綠所 XIULU

> 地方創生團隊 × 鐵窗花風 Iron-Grille
> Design Skills Center 館藏 Vol.02

一支在嘉義、台南、彰化做老屋鐵窗花田野普查與再製的地方創生工作室。網站不擺型錄要你挑，
而是把最核心的那件事——**一面窗要焊得起來，每一根鐵桿都得接得到隔壁的桿**——寫成一支
**波函數塌縮（Wave Function Collapse）** 引擎搬到首屏：你釘住幾格母題，約束沿著相鄰的縫傳播，
替你把整面一致、可焊接的窗補完。

## 認領的首創

**波函數塌縮（WFC）約束傳播作為互動範式與全站唯一圖像引擎。**

每一格窗一開始都「還沒決定」，同時保有全部 41 種母題變體。使用者釘住一格＝在觀察前先替某格定案，
約束沿四條縫傳播、把不可焊接的選項逐一刪掉，直到整面窗只剩一種一致解——它**保證焊得起來**。
撞牆（某格選項歸零）時換種子重跑（上限 40 次），條件互相矛盾則以保底解收場並如實呈現。
同一支引擎渲染：設計檯即時窗、24 面普查圖鑑、logo、favicon、每張受理單的窗籍印——**全站無一張圖是「畫」的**。

與館內既有生成／規則類首創的差異：`dice-needle` 是 seeded PRNG 直接生成、無約束傳播與回溯；
`shaft-eight` 的織法引擎是經緯浮沉矩陣、無熵與觀察順序；`foldpoint` 是剛性摺疊運動學；
`kinrin-en` 是二倍體遺傳；`qizhong` 是排列群步行；`muli-arbor` 是逆問題反演。
本站是第一個以**約束滿足＋熵導向塌縮＋回溯**為核心互動的站——你不是在畫圖，是在下條件，讓規矩把窗長出來。

## 核心功能（§2.4-7 function 維度）

**客製化配置器（選材質顏色即時預覽）**。窗格數 × 對稱 × 母題組 × 配色 × 母題筆釘選 → 塌縮出一面窗 →
即時 BOM（桿長含外框、焊點含母題內／內縫／框格三類、花面片數、成品尺寸）→ 花面材質（純鏤空／彩色玻璃／壓克力）、
表面處理（古法手刷漆／戶外粉體烤漆／熱浸鍍鋅）、常規／急件 → 分項報價與工期 →
四欄驗證（稱呼 ≥2 字、手機 09 十碼、安裝地址 ≥6 字、Email 選填格式）生成 `XL-YYMMDD-NNN`
受理單與由單號雜湊出的窗籍印。設計可用 `?w=` 分享碼序列化重現。

## 技術

單檔 HTML，CSS 與 JS 全部 inline。零外部圖片、零外部音檔、零第三方函式庫、零 WebGL／Three.js
（WFC 引擎與 SVG 渲染全部自寫）。外部資源僅 Google Fonts。favicon 為原創 inline SVG data URI。

`prefers-reduced-motion` 下取消所有過渡，受理單不平滑捲動；塌縮、釘選、篩選、試算、表單等全部邏輯完整保留
（本站無自動播放動畫，會動的只有使用者操作與塌縮重繪）。無 JavaScript 時：`zhi.html` 的 24 面普查圖為建置時
烘好的靜態 SVG 完整可見、四條件內容與費用表為靜態 HTML，`gongfa.html` 全頁靜態可讀，`index.html` 以
`<noscript>` 指向窗誌、工法與放樣專線。
