# 余白 YOHAKU 瑜伽 — 極簡留白（Minimalist Negative Space）

> Design Skills Center 館藏範例站 · 產業 × 風格：**瑜伽教室 × 極簡留白**

一間虛構的台北瑜伽教室。以「留白，是刻意留下的空」為主軸，整站示範**極簡留白（negative-space minimalism）**：極大量的空、極細的髮絲分隔線、幾乎單色的紙灰調、輕量無襯線與細襯線的疏密對比，以及一個隨呼吸緩慢張縮的圓環作為動效簽名。首頁採「無 hero・章節卷軸」開場，導覽是頁側章節錨點索引。

## 風格關鍵字

極簡留白、negative space、大量空白、髮絲線分隔、近單色紙灰調、輕量無襯線（Jost）＋細襯線（Noto Serif TC 200）、疏密對比、章節卷軸（chapter-scroll）、錨點索引導覽（anchor-index）、呼吸節拍圓環、無圓角、零陰影、緩慢淡入。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連，共用一致的導覽列與 footer。
- 所有圖像為原創 inline SVG（呼吸圓環、教室線稿、老師肖像剪影、路線圖），零外部圖片；外部資源僅 Google Fonts（Jost + Noto Serif TC）。
- favicon 為原創 inline SVG data URI（同心圓＋石板藍核心）；`assets/logo.svg` 為原創字標。
- 互動：行動選單、呼吸圓環（吸／停／吐 節拍）、章節錨點 active 高亮、進場淡入揭示，全部支援 `prefers-reduced-motion` 降級（圓環停在靜態）與 RWD（≤560px 不破版）。本站不使用跑馬燈——動效簽名是「呼吸節拍圓環」。
