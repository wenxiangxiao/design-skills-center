# RASTER 當代美術館 — 瑞士國際主義（Swiss International Style）

> Design Skills Center 館藏範例站 · 產業 × 風格：**美術館展覽 × 瑞士國際主義**

一座虛構的台北當代美術館。以「系統、網格與偶然」為策展主軸，整站示範**瑞士國際主義／國際主義排版風格（International Typographic Style）**：嚴格的欄位網格、齊左不齊右、無襯線大字、紅黑白三色、以幾何構成取代攝影插圖。

## 風格關鍵字

瑞士國際主義、國際主義排版、Josef Müller-Brockmann、Max Bill、Akzidenz-Grotesk、數學網格、齊左不齊右、紅黑白、客觀性、幾何構成、無圓角、細硬分隔線、編號系統。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連。
- 所有圖像為原創 inline SVG（海報、縮圖、地圖、平面圖），零外部圖片；外部資源僅 Google Fonts（Archivo + Noto Sans TC）。
- favicon 為原創 inline SVG data URI；`assets/logo.svg` 為原創字標。
- 互動：行動選單、跑馬燈、進場揭示、展覽篩選、票價試算，全部支援 `prefers-reduced-motion` 降級與 RWD（≤560px 不破版）。
