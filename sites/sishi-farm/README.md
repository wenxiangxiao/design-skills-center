# 四時直送 SÌSHÍ — 自然有機風（Natural Organic）

> Design Skills Center 館藏範例站 · 產業 × 風格：**農產品直售 × 自然有機風**

一個虛構的南投埔里小農產地直送品牌。它的核心主張是「只賣當令」，所以整個網站的首頁不是大標開場，而是一張**會轉的產季曆（schedule-first）**：一個可點選的四季轉盤，轉到當下的節氣，就看見田裡真正在採收的作物。導覽是一條固定在左側的**綠色側欄（side-rail）**，像一本農民曆的書背。整站示範自然有機風——亞麻米底、森墨綠與陶土大地色、手繪蔬果線稿、葉脈般的描線生長動畫、紙張顆粒質感、波浪分隔線。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連，共用一致的 side-rail 導覽與 footer。
- 所有圖像為原創 inline SVG（四季轉盤、蔬果線稿、農友肖像、波浪線）與 CSS 紙張顆粒質感（feTurbulence data URI），零外部圖片；外部資源僅 Google Fonts（Newsreader + Noto Serif TC + DM Sans）。
- favicon 為原創 inline SVG data URI；`assets/logo.svg` 為原創四季環字標。
- 互動：四季產季轉盤（點選→指針旋轉＋面板切換＋葉脈線稿重繪）、依當前月份自動預設季節、產地目錄，全部支援 `prefers-reduced-motion` 降級（關閉指針旋轉與描線動畫）與 RWD（≤820px 側欄轉為頂部橫向列，不破版）。
- 刻意**不使用跑馬燈**：以「可旋轉的產季轉盤＋葉脈生長描線＋左側農民曆側欄」作為本站的識別性版面與互動簽名。
- 敏感性：本站所有品牌、人物、地址、價格皆為虛構示意。
