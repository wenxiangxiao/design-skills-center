> 一所研習與保存舊式電碼、譯電技術的傳習所。這個網站的正文，一開始是讀不出來的——它以本所的減碼本加密過，你得先把減碼推對。

**產業 × 風格：** 譯電傳習所 × 打孔帶格紙風 Punchtape-Grid
**線上首創：** 疊加式數字密碼（電碼＋減碼本 subtractor）的密碼分析作為互動範式，且解出的減碼會套用到整座檔案室。

## 品牌設定

碼房是一所還在上課的傳習所，不是靜態博物館。它收整廢線上的殘電、整理老碼本、開班教人譯電。語氣是嚴肅職人：話少、務實、行話從電報產業裡長出來（明碼、電碼、減碼、抄收、校紅、存查、拍發、波多紙帶）。地址、電話、班別、費用皆為虛構教學示意。

## 頁面

1. **index.html — 譯電檯**：首屏即一封尚未譯出的內電（ciphergram-first 開場）。轉動減碼盤、以情報便箋上的已知字回推減碼，讓每一群電碼一格一格從紅（■）翻綠成明碼。全單譯出後給受理章與證明，並把減碼存入本機作業階段。
2. **codebook.html — 碼本查號**：公開電碼本（約 260 字），依部類／筆畫／關鍵字查號；另以逐位運算表講解「明碼→電碼→減碼」三層與破解的逆運算。

- **opening：** ciphergram-first 密電待譯開場（新原型）—— 首屏即整頁待解電文＋減碼盤，明碼須解出才浮現；附靜態明碼與人工譯電管道保底。

## 首創差異說明

- 與 **muli-arbor**（逆問題）不同：那是連續二維真值場＋SIRT 反演、使用者設計量測幾何；本站真值是一段離散、確定、有限的文字，靠還原 key 而非幾何量測取得，且以「碼本合法性＋已知字回推的深度利用」為破解手段。
- 與 **unknown-field 開場**不同：那裡尚無資料、需量測才產生；本站資料齊全，只是被加密——解密即讀出。
- 與 **transcript-gap 開場**不同：那份筆錄本來就完整可讀、只缺一句；本站首屏整頁不可讀，須解密才成形（附靜態明碼備援，資訊仍可尋得）。
