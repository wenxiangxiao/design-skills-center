**產業：** 手工絞染工坊　**風格：** 摺染紋樣風 Shibori-Symmetry　**語言：** 繁體中文

台南後壁菁寮老街上，林淑貞師傅的手工植物絞染工坊。這座線上染台不給你現成的圖案——你先選一種**摺法**，摺法就決定了整幅布的**對稱**；再下型板夾、入茜缸，**解開**時你在一小疊布上動的那一角，會被摺法的對稱鋪滿整幅布。

## 核心互動（本站首創）

**摺疊對稱防染引擎（fold-resist symmetry dyeing engine）。** 全館第一個以「摺疊對稱防染」為核心互動的站：

- **摺法給對稱群**——三角雪花摺 = 二面體 D8 放射對稱；板締格摺 = 上下左右鏡射網格；嵐斜捲 = 斜向滑動鏡射。
- **型板給基本域**——你選的紋樣只作用在一枚楔形／一小格「基本域」上。
- **染料給邊界條件**——下缸深淺決定暈邊寬窄；套染疊第二色。
- **解開 = 對稱群作用在基本域上**——同一組型板，換一個摺法就成了完全不同的滿幅樣，因為換的是對稱本身。

網站上每一塊布、logo、favicon、圖鑑二十四幅、訂單絞印，都由這同一支引擎解出，**沒有一張是描的線、沒有一張外部圖片**。

## 首創登記

摺疊對稱防染（fold-resist symmetry dyeing）作為互動範式與全站唯一圖像引擎。與 shaft-eight（經緯浮沉矩陣渲染布面、無對稱群與防染）、foldpoint（剛性摺疊運動學）、xiulu（WFC 約束傳播）、kinrin-en（遺傳世代）、leung-kee（真實時刻晝夜引擎驅動的街景霓虹「顯示」）皆不同：本站互動對象是一塊布的摺疊對稱群與其防染邊界條件，圖案是對稱群作用在使用者防染基本域上的解。
