## 版面原型（Layout Prototype）

- **opening = toc-first**：首頁第一屏就是服務目錄——一整面密排的價目招牌牆（每項服務＋價格），本身即是可用的選單／索引，而非大字 hero 標語。
- **nav = topbar**：頂部一列做成琺瑯牌面分頁（首頁・價目表／老店故事／預約），行動裝置（≤560px）自動換行收合。
- **signature = 旋轉三色燈柱 + 招牌翻牌**：用 `repeating-linear-gradient` 加 `background-position` 動畫做無限向上捲動的紅／白／藍理髮燈柱（hero、nav、footer 皆有）；價目招牌以 CSS 3D `rotateY` hover 翻面，背面露出一句台語俏皮話。`prefers-reduced-motion` 下燈柱靜止、翻牌停用只顯示正面。
- **illust = sign-lettering 手繪招牌美術字／琺瑯色塊**：所有圖像皆為原創 inline SVG，由實色色塊構成（琺瑯招牌、燈柱、剪刀、龍鳳字牌、示意地圖），無任何外部圖片、無細線 lineart。
