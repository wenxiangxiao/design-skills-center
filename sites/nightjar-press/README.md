# 夜鴞出版社 Nightjar Press — 暗黑學院風（Dark Academia）

> Design Skills Center 館藏範例站 · 產業 × 風格：**獨立出版 × 暗黑學院風**

一間虛構的台北獨立出版社，藏在溫州街舊公寓二樓，只在深夜校稿。整站示範 **暗黑學院風（Dark Academia）**：暖墨深底、羊皮卷米白、燭火般的黃銅金與牛血紅，全站襯線字體、首字放大、細金分隔線、凸版與版畫感的原創插圖，動效緩慢而克制——燭火微顫、進場揭示、書脊抽出。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連。
- 所有圖像為原創 inline SVG（夜鴞版畫、燭台、書封、肖像、書系符號），零外部圖片；外部資源僅 Google Fonts。
- favicon 為原創 inline SVG data URI（金色弦月＋微星）；`assets/logo.svg` 為原創圓形社徽字標。
- 互動：行動選單、燭火微顫、進場揭示、書脊 hover 抽出、書系篩選、訂閱回饋，全部支援 `prefers-reduced-motion` 降級與 RWD（≤560px 不破版）。
- **無跑馬燈**：本站以「書脊書架」與「進場揭示」承擔動效簽名，刻意不使用捲動橫幅（見 PLAYBOOK §2.1）。
