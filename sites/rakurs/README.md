# 銳角桌遊社 RAKURS

**產業**：桌遊出版　·　**風格**：構成主義 Constructivism

一間虛構的台北桌上遊戲出版社，把構成主義當成設計憲法：只有紅、黑、鋼藍、赭黃，只有 31 度的斜軸與硬邊。首頁是一張滿版的構成主義海報，捲動時幾何色塊沿斜軸滑入拼合；遊戲庫是一座可多條件即時篩選的資料庫，把不合的盒子推開，只留今晚該上桌的那幾款。
