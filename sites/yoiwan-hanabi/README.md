**有限聲速的聲光時差作為互動範式（finite sound-speed light/sound offset as a vantage-bound experience）**：使用者選定觀覽座位後，整場番組以物理正確的「光即時、聲後到」從該座位重演——遠席先見無聲綻放、約 `距離/340` 秒後爆音與提灯震動才抵達；座位改變的不只是視角縮放與方位，還有每一發的**聲音到達時刻**，故遠近席聽到的節奏完全不同。與 flicker-house「選座即見該席望向銀幕的視線預覽」的差別：那是靜態幾何視線、無時間維度；本站的核心是**時間**——同一發花火在不同席位以不同時刻抵達耳朵。與 tide-signal（即時音訊合成）、leung-kee（真實時刻晝夜引擎）的差別：那些不模擬空間造成的知覺延遲。
