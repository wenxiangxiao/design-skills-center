## 頁面

1. `index.html` — 路網首頁。**boot-loader 點火開場**（系統自檢 → 轉動鑰匙）後，進入可駕駛的俯視小鎮：方向鍵開車、大燈掃光、駛近建築即亮方向燈並提示「按 Enter 進入」。手機顯示螢幕 D-pad，建築也可直接點按；`prefers-reduced-motion` 下跳過點火動畫、建築即為純連結。
2. `courses.html` — 課程與費用。四張號誌卡（班別／堂數／費用）＋四班對照表＋報名須知路標。

## 首創說明（登記於 ledger `_first_registry`）

**鍵盤駕駛導覽（Drive-to-navigate）**——本站是館內第一個「開車逛的網站」：首頁不是靜態版面，而是一台可用方向鍵／WASD 駕駛的小車與一座俯視路網。導覽動作＝駕駛動作，開進黃棟看資訊、開進紅棟去報名。實作為 `requestAnimationFrame` 的極簡車輛模型（加速度＋摩擦＋隨速轉向），駛入建築的鄰近圈即高亮該建築並提示 Enter 進入。符合 PLAYBOOK §2.6 破格條款「打破網站慣例但仍可用」：底部儀表板恆列各頁直達鍵、建築本身即連結、手機提供 D-pad、`prefers-reduced-motion` 降級為純連結地圖，資訊永遠找得到。
