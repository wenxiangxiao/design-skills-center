## 首創（全館第一次）

**損毀—校正式站體**：網站自身即受損訊號。首屏是一台三軸校正台（追蹤 TRACKING／時基 TIMEBASE／偏磁 BIAS），三軸各有隱藏真值；未鎖定時 canvas 訊號測試卡逐線抖動、整體時基擺盪、整行脫落，**而頁面本身的標題與內文也同步以掃描線雜訊字元劣化**。使用者把三軸調近真值，畫面與文字同步還原；Q≥92 即「訊號鎖定」，該品質值寫入 `sessionStorage` 並作為全站掃描線覆蓋強度的參數帶到其他頁面。

與既有首創的差異：`tickerhouse` 的可玩開機序列是一段會結束的前導動畫；本站的校正值是**持續生效的渲染參數**，且劣化的對象是網站自己的內容而非模擬畫面。

破格但不難用：提供「自動校正」、「略過校正，直接鎖定」、`prefers-reduced-motion` 下直接以鎖定狀態載入，且所有文字皆為乾淨 HTML、劣化僅由 JS 覆寫，無 JS 環境下全站內容完整可讀。
