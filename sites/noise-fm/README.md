# 雜訊電台 NOISE FM — 新粗獷主義（Neo-brutalism）

> Design Skills Center 館藏範例站 · 產業 × 風格：**Podcast 節目 × 新粗獷主義**

一檔虛構的中文網路文化 Podcast。以「訊號裡的雜訊，放大給你聽」為主軸，整站示範**新粗獷主義（Neo-brutalism / web brutalism）**：厚實 3px 黑框、硬邊位移陰影（`6px 6px 0`）、高飽和撞色（芥末黃／鈷藍／番茄紅／萊姆綠）、Archivo Black 巨體＋Space Mono 等寬字，以及外露、方正、不修飾的結構。首頁採「產品網格開場（catalog-first）」——一進站就是一整牆的節目卡；導覽是厚框置頂列。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連，共用一致的置頂導覽列與深色 footer。
- 所有圖像為原創 inline SVG／CSS（等化器條 logo、波形、主持人剪影、播放三角），零外部圖片；外部資源僅 Google Fonts（Archivo Black + Space Mono + Noto Sans TC）。
- favicon 為原創 inline SVG data URI（黃底黑框＋等化器條）；`assets/logo.svg` 為原創字標。
- 互動：行動選單、**硬陰影方塊按壓位移**（卡片／按鈕 hover 時 `translate(4px,4px)` 貼向陰影）、**節目數據翻牌計數器**（進場由 00 累加到目標值）、示意播放器（播放鈕切換＋波形跳動＋計時），全部支援 `prefers-reduced-motion` 降級（取消位移與跳動）與 RWD（≤560px 不破版）。
- 首頁頂部為**靜態**資訊帶（非捲動跑馬燈），動效簽名由「按壓位移」與「翻牌計數器」承擔，避免落入跑馬燈反射式套用。
