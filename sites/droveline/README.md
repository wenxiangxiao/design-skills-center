# 牧線工作站 DROVELINE

宜蘭三星行健溪河灘牧地的邊境牧羊犬行為訓練工作站。

**產業 × 風格：** 牧羊犬訓練場 × 軌跡疊印風 Trace-Overlay

## 這個站在做什麼

首頁不是介紹頁，是一片**正在發生的牧地**。八隻羊自己在吃草、自己會聚群、自己會怕犬；犬趴在牧人樁旁邊等哨。
你不能拖動任何一隻羊，也不能拖動犬——**你只能吹哨**。哨令進入犬的隊列後要等牠自己的反應延遲（0.26–0.62 秒，逐犬不同）
才會生效，然後犬照牠自己的走法執行，羊再照牠們自己的規則反應。三層裡沒有一層是你直接操作的。

走完一趟四段賽程（取羊／帶回／驅趕／入欄）之後，系統不是給你一枚分數徽章，而是分析**你這雙手的哨令習慣**
（指令密度、用距、停桿比例、穩桿比例、繞邊偏向、羊群散開度），拿去比對十二隻犬各自吃得下的手，
配出前三名並附上落差說明——然後給你一顆按鈕：**換那隻犬再走一趟**。
新的軌跡會疊在同一塊版子上，配對準不準，你自己看得出來。

## 首創（全館第一次）

**間接控制的雙層自主系統。**

全館既有的互動站，使用者操作的對象都聽自己的話：gluhwerk 的玻璃隨你變形、jinsheng 的戲偶跟著游標、
tackline 的船舵指哪走哪（與你作對的是環境）、nodal-luthier 照你刨的厚度回答、foldpoint 照你拖的角度摺、
meixi-signalbox 會拒絕你但拒絕之後仍然照規則執行、yongzi-foundry 照你拉的桿鑄字。

本站是第一個**使用者不能操作任何一個對象**的站：

1. 你只能對一個**代理**（犬）下達有限詞彙的指令，
2. 該代理有自己的反應延遲與執行邏輯，指令送出後不可撤回，
3. 代理再以壓力場影響**第二層自主群體**（八隻各自跑 boids 規則的羊），
4. 而你的目標只能透過這條間接鏈達成。

並且——這條指令鏈本身就是診斷的輸入：系統分析的不是你的成績，是**你下令的語言習慣**，
再把它翻譯成「哪一隻犬吃得下你這雙手」的可驗證假設。

## 風格關鍵字

單色墨階、靛藍、暗場亮跡、累積曝光、軌跡即圖像、量測台、無圓角、無陰影、細線與粗淡線的明度分工、
窄體標題（Antonio）、表格式資訊、不對稱走道版面。

## 技術

- 四頁單檔 HTML，CSS／JS 全 inline，頁面間相對路徑互連
- 零外部圖片、零外部音檔、零第三方函式庫、零 WebGL、零 3D
- 外部資源僅 Google Fonts（Antonio + Archivo + Noto Sans TC）
- favicon 為原創 inline SVG data URI
- `prefers-reduced-motion`：不啟動 rAF，改以「單步 +2 秒」與「示範走場」取得完整成績與診斷，模擬邏輯完整保留
- 無 JavaScript：四頁的說明、規則、算式、犬籍、課表、費用全部是靜態 HTML，另附 `<noscript>` 替代管道
- 十二張軌跡肖像是**建置時**由同一支模擬引擎跑出來後烘成 SVG 的，不需要 JS 也看得到
