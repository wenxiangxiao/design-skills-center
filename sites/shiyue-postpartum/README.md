台中「拾月堂產後護理之家」為**虛構品牌**，本站為設計範例展示：以「層疊剪紙／紙雕」美學打造的月子中心官網，含 schedule-first 月子餐週表開場、房型與照護頁，以及完整可玩的「入住試算器」。

> 本站內容為虛構示意，僅供設計範例展示，非真實營業機構。

## 硬性輪替約束落實

- **開場原型 schedule-first**：首頁第一屏即「本週月子餐時刻表」——七天 × 三餐兩點心的剪紙看板，可切換本週／下週，點任一餐點展開食材與功效。
- **導覽原型 topbar**：頂欄造型為剪紙屋簷簷口，雙層圓齒刀花邊（前層霧綠＋後層深霧綠紙厚陰影）。
- **插畫技法**：全站 SVG 皆為層疊剪紙——每層純色剪影，下一層以同色系深色 offset 營造紙層厚度；無漸層、無線描、無暈染。
- **Mood pastel-soft**：暖杏 `#F6DFD3`、霧綠 `#CFE0D5`、藕粉 `#EBC7C2` 三色粉彩底，無米白紙感、無深底。
- **功能**：入住試算器——預產期＋生產方式 → 建議入住／離所日；三房型（每日價）＋ 14–40 天滑桿 → 即時總價、90 天早鳥 95 折、10% 訂金；一鍵產生可複製的「試算單」摘要卡。

## 設計要點

- 字體：Noto Serif TC 600（標題）＋ Noto Sans TC（內文），僅載入 Google Fonts，零外部圖片。
- 每頁單檔 HTML（CSS／JS inline），相對路徑互連，共用簷口 nav 與 footer。
- Favicon 為原創 inline SVG data URI（剪紙彎月）。
- 區塊交界使用剪紙波浪與圓齒刀花分隔；卡片以「紙層」語彙（細 offset 邊＋柔和環境影）呈現。
- `prefers-reduced-motion` 降級：雲朵飄動與過場全部停用；RWD 至 360px 寬不破版，週表於窄幅改為橫向滑動看板。
