## 首創（全館第一次）

**織法引擎作為全站唯一渲染器**——使用者編輯的四張表是一段離散程式（穿綜／綁踏／踏序／色序），引擎把它編譯成 32×32 交織矩陣，再逐紗渲染成布面；同一支引擎同時產出使用者的作品、24 張布樣縮圖、全站六張插圖、logo、favicon 與訂單的布邊識別紋。**本站沒有任何一張圖是「畫」的**，全部是織出來的。引擎另負責結構驗證：浮長、交織率、不織經紗、從未提起的綜框——使用者寫的程式會被回頭檢查能不能變成一塊撐得住的布。

與館內既有首創的區別：`gluhwerk-glass`／`jinsheng-glove`／`tackline` 操作的是連續物理量；`grip-station` 是使用者擺位置、系統回頭演算；`nodal-luthier` 是操作看不見的參數由物理場作答。本站是**離散組合程式 → 材料**，而且工具本身就是全站的美術資產產生器——換掉引擎，這個網站會變成空白。
