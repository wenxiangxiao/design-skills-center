# 毫巴潛水學校 MILLIBAR

**產業**：潛水學校（綠島）　·　**風格**：氣象圖／等壓線美學

一間開在綠島南寮的潛水學校，把「讀天氣」當成教潛水的第一步。整站是一台綜觀海況控制台——石板墨藍的等壓線、風羽、站點模式，全部原創以 canvas 生成。首頁不是漂亮的水下照片，而是一張會動的天氣圖：拖動預報時間軸，等壓線即時重繪，湧浪、能見度與適潛指數同步更新。

## 八維配方

> **潛水學校 × 氣象圖／等壓線美學 × 近未來（資料驅動控制台）× 台灣本土（綠島）× 冷靜理性 × 金屬（儀表面板）× 中等密度 × 預約／訂位系統**

- **era**：近未來——像一台海象監測儀表台，資料驅動、無裝飾。
- **locale**：台灣本土——綠島南寮、石朗、柴口、朝日溫泉、討海船長。
- **voice**：冷靜理性——用數據說話、不販賣冒險感，把安全講在最前面。
- **texture**：金屬——儀表面板、傳真機、鉻色滑桿的冷硬手感。
- **density**：中等——控制台資訊密，內文段落留白。
- **function（核心可玩功能）**：**預約／訂位系統**——選日期（每天附模擬適潛指數）→ 選潛點（依當日海況顯示「適合／不建議」）→ 選課程與人數 → 右側即時結算海況與總價 → 送出取得訂位代碼；指數過低自動標記改期。

- `index.html`　海況氣象台（os-metaphor）：整頁做成 MILLIBAR 綜觀海況控制台。canvas 以高低壓高斯場 + marching squares 生成等壓線、風羽與 H/L 中心；拖動預報時間軸讓整張圖重繪、讀值面板與適潛指數連動。下接看圖三準則與課程一覽。
- `book.html`　預約下水（核心功能）：日期卡片各帶適潛指數條、潛點依海況顯示適合與否、課程與人數選擇、右側 sticky 摘要即時結算，送出生成 `MB-YYMMDD-XXX` 訂位代碼與訂單存根。
- `about.html`　關於・安全簡報：品牌來歷、四條安全準則、原創綠島潛點示意地圖（疊於等壓線）、教練群與統計。

## 版面原型

- **opening**：os-metaphor（綜觀海況控制台介面隱喻）
- **nav**：side-rail（左側垂直儀表側欄，手機收合為頂列 tab）
- **illust**：generative-canvas（canvas 生成等壓線場、風羽、站點模式）
- **mood**：mono-ink（石板墨藍單色墨階 + 單一低壓警示紅）
- **signature**：拖動預報時間軸使 canvas 等壓線場即時重繪（高低壓中心飄移、線密度改變），風羽方向與湧浪／能見度／中心氣壓／適潛指數同步更新；低壓中心以紅色標記
- **func**：預約／訂位系統（選日期看指數 → 選潛點看適合度 → 選課程人數 → 結算 → 訂位代碼）
