## 開場與導覽原型

- **開場（opening）**：`map-first` — 首頁第一屏就是可互動的街廓地圖與物件清單，把「先看在哪、再談價格」當成主敘事，而非置中大標。

- **動效簽名（signature）**：街廓地圖圖釘 hover 連動右側物件卡雙向高亮＋紅色虛線十字準星對位＋總價數字滾動計數（easing 緩動）＋瑞士 2px 網格基線與模組化物件卡。全館其他站沒有「地圖 × 物件清單雙向連動」這一招。
