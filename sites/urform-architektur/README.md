## 技術說明

- 三頁皆為**單檔 HTML**，CSS/JS 全 inline，頁面間以相對路徑互連，共用一致的導覽列與 footer。
- 字體僅用 Google Fonts：`Poppins`（幾何無襯線，標題／標籤）＋ `Noto Sans TC`（中文內文）。
- **零外部圖片**：所有視覺為原創 inline SVG（幾何構成、建築抽象縮圖、logo、地圖）；favicon 為原創 inline SVG data URI 寫在每頁 `<head>`。icon 全自繪，無 emoji。
- 招牌互動：首頁「幾何構成器」以純 JS 重組圓／方／三角與三原色，產生不同海報構成（不使用跑馬燈）。
- 動效：幾何進場揭示（IntersectionObserver）、緩慢旋轉的線框圓、hover 色塊翻轉；全部提供 `@media (prefers-reduced-motion: reduce)` 降級。
- RWD：≤960px 版面收合，≤560px 提供行動選單，手機寬度不破版。
- 風格規格見 `SKILL.md`（`name: bauhaus-primary`），該規格不綁定「建築」題材，可套用到任意產業。
