## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連。
- 所有圖像為原創 inline SVG／CSS 漸層（Logo、壓軸卡底圖、火花、星形），零外部圖片；外部資源僅 Google Fonts（Chango + Orbitron + Noto Sans TC）。
- favicon 為原創 inline SVG data URI；`assets/logo.svg` 為原創鉻金屬字標。
- 互動：行動選單、情緒環隨機色、票數即時試算、日期切換時刻表、游標星塵拖尾，全部支援 `prefers-reduced-motion` 降級與 RWD（≤560px 不破版）。
- 刻意不使用跑馬燈：以「桌面視窗版面＋游標星塵拖尾＋情緒環」作為本站的動效簽名。
