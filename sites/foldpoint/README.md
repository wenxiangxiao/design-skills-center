## 認領的首創

**剛性摺疊運動學作為版面容器**——平面刀模與立體成品是同一組資料的兩個投影，摺疊角 0–100% 連續且可逆。

一組面片以「父面片 ＋ 摺線軸 ＋ 二面角」構成樹狀結構，每一片的世界矩陣為
`M_child = M_parent · Rot(hinge, φ)`，摺疊度 `t` 只是把所有 φ 乘上同一個係數。
於是 **`t=0` 畫出來的就是可送模切廠的展開圖，`t=1` 畫出來的就是成品**，中間任何一格都是真的中間狀態，
不是兩張圖之間的補間動畫。摺線的旋轉方向由面片重心相對摺線軸的外積符號自動判定，不需人工標註山谷摺。

同一支引擎另外承擔：導覽（一個方管，四個面就是四個頁面，頁名以面片投影解出的 `matrix()` 貼在面上）、
盒型庫 24 張縮圖、落數排版圖的單件外框、以及每張報價單的糊耳暗紋。

與館內既有 3D／空間類首創的差異：ridgeline／compositie-optiek 是 Three.js 場景與商品、
flicker-house 是 CSS 3D 的固定放映廳、shaft-eight 是離散織法矩陣渲染布面——
本站是第一個「同一份幾何同時是 2D 工程圖與 3D 物件，且使用者可以在兩者之間連續拖動」的站。

## 核心功能（§2.4-7 function 維度）

**試算／計算器：開紙落數與紙材用量**。輸入盒型碼與數量 → 展開尺寸 → 在全開紙上扣掉咬口 10mm、
四邊留邊 8mm、刀間距 3mm 後以兩種取向排版取較優解 → 落數／邊料比例／放損（200 張＋3%）／用紙張數 →
紙價（依張面積 × 磅數 × 每公斤單價）、印刷（紙板與 E 楞走平版、B／AB 楞走水性柔版）、
裱合、刀模費（依展開面積分級）、模切工、表面加工、糊工、分裝、急件加成 → 單價與交期。
表單四欄驗證後生成 `FP-YYMMDD-NNN` 報價單與由單號雜湊出的糊耳暗紋。

## 技術

單檔 HTML，CSS 與 JS 全部 inline。零外部圖片、零外部音檔、零第三方函式庫、零 WebGL／Three.js
（3D 由本站自寫的 3×4 矩陣與 Rodrigues 旋轉直接投影成 SVG 多邊形），外部資源僅 Google Fonts。
favicon 為原創 inline SVG data URI。

`prefers-reduced-motion` 下導覽方管不做慣性旋轉、自動摺直接跳到成品、報價單不平滑捲動；
摺疊桿、視角、試算與篩選等全部邏輯完整保留。無 JavaScript 時四頁文案、規則、費用表與問答皆為靜態
HTML 可讀，並以 `<noscript>` 附上電話與替代取得方式。
