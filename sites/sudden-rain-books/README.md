# 驟雨書店《驟雨日報》

基隆獨立書店「驟雨書店」的範例網站。整個網站做成一份報紙——《驟雨日報》：
報頭、分欄、雙線框、首字放大、圖說、分類廣告、版權欄，一應俱全。
所有插畫為單色木刻版畫風的原創 SVG，僅外連 Google Fonts，其餘資源全部內嵌。

## 檔案結構

```
sudden-rain-books/
├── index.html        頭版・要聞——本月選書專題（頭條）、店內動態四則、天氣欄、本期要目、老派廣告框
├── shelves.html      二版・書架——五座書架十七種選書，每本附「重刻版」SVG 書影與店主短評
├── about.html        三版・副刊——店主專欄長文〈守店記〉、書店年表、來店資訊＋版畫地圖、分類廣告
├── assets/
│   └── logo.svg      原創報頭 logo（雨中燈塔報徽＋刊名＋英文副題）
├── README.md         本檔
└── SKILL.md          「報紙編輯風」設計規格書，供任何 AI 或設計者重現此風格
```

## 使用方式

無需建置、無需伺服器，直接以瀏覽器開啟 `index.html` 即可。
三頁之間以相對路徑互連（頭版・要聞／二版・書架／三版・副刊）。

## 設計重點

- **一報三版**：網站結構即報紙結構——頭版（頭條＋簡訊＋邊欄）、二版（書單）、三版（副刊＋分類廣告）。
- **報頭 masthead**：報眉（發行地＋即時日期＋天氣一行）、報徽、刊名、英文副題、期數欄（folio）、版面索引導覽，三頁完全一致。
- **色彩**：紙色底（#f4eede）、油墨黑（#221d15），唯一強調色為印章紅（#a63b2c），用於 kicker、當前版面、紅印與分類標籤。
- **字體**：Noto Serif TC（全字重變數字型，內文 400、標題 900）＋ Playfair Display（英文副題與書名原文，斜體）。
- **排版**：CSS multi-column（頭條三欄、專欄兩欄）＋ 1px 分欄線；標題階層以字重與字距區分；`::first-letter` drop cap；雙線（double）框用於天氣欄、廣告、引言與各種「重要之框」。
- **插畫**：全部為手刻的 inline SVG——報徽燈塔、內港版畫、十七張書影、港邊地圖、天氣雲，統一單色線刻＋平塗黑＋排線（hatching）畫法，紅印落款。
- **動效克制**：進場淡入（IntersectionObserver）、書影 hover 微傾、報眉日期以 JS 即時顯示（民國紀年＋星期）；`prefers-reduced-motion` 時全部降級為靜態。
- **RWD**：960px 以下邊欄落下、多欄減欄；700px 以下收為單欄，仍保留雙線框、分欄線與報頭質感。

## 內容說明

店名、店主、店貓、地址、電話、活動與《雨港街路考》《崁仔頂魚市誌》兩種「本店寄售」書目為虛構；
其餘選書（如黃錦樹《雨》、廖鴻基《討海人》、鄭栗兒與鄭順聰《基隆的氣味》等）為真實出版品，
定價以版權頁常見定價擬寫，僅供範例展示，購書請以各出版社現行定價為準。
