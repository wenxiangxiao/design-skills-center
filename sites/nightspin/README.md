# 夜滾自助洗衣 Nightspin Launderette

**產業 × 風格**：自助洗衣店 × 夜光投幣機風（Coin-Op Nightshift）

台北大同區赤峰街尾一棟老公寓一樓的 24 小時無人自助洗衣店。凌晨三點，燈管嗡嗡亮著，滾筒轉著別人的衣服，公告欄上釘著一整晚陌生人留下的話。本站把「深夜一個人洗衣、卻與陌生人非同步共用一個公共空間」的孤獨與陪伴，做成一個照著你現在幾點即時運轉的介面。

## 核心功能（可玩）

**樓面選機台（票務選位型）**：點一台亮綠燈的空機 → 選洗程／水溫／衣量 → 即時試算時間、費用與投幣枚數 → 投幣認領 → 取得倒數領洗票。認領狀態寫入 `localStorage`，重整後倒數繼續、洗好會提醒收衣；順手留的紙條會釘上公告欄並保留。

## 首創（全館登記）

**偽即時多人共同在場**（fake-live co-presence／asynchronous social traces）：以 `f(machineId ^ 當日種子, 當地時鐘)` 的 seeded PRNG 為每台機器生成當日排程並跨午夜 spillover，讓畫面照真實時刻呈現「哪台在轉、誰洗好了還沒收」；夜間動態 ticker 由所有機台近 50 分鐘的起訖事件即時彙整；公告欄＝seeded 他人紙條＋使用者持久化紙條混排。使用者的認領與紙條跨重整存活並融入集體，於是「其他人」既非你能操作的對象、也不是自主物理群體，而是一批非同步共用同一公共空間的陌生人。與 droveline／live-world 的差別：那些是你觀察或指揮的自主物理代理，本站模擬的是社會性的人的痕跡與在場感。
