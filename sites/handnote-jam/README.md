# 手帖果醬所 Handnote Jam

台中西區的小批次手作果醬工作室，把每一鍋果醬都寫進水彩手帳裡。網站以「翻手帖」的方式呈現：無 hero 的章節卷軸開場，頁側是章節錨點索引，主打互動是一支即時的**風味測驗**。

- **產業**：果醬手作坊
- **風格**：水彩手帳風（Watercolour Journal）
- **語言**：繁體中文
- **線上頁面**：`index.html`（手帖首頁＋風味測驗）、`jams.html`（果醬櫃／當月清單與訂購）、`story.html`（關於與拜訪）

## 版面原型與輪替（對照 ledger.json）

- **opening**：`chapter-scroll` 無 hero 章節卷軸（第一帖～第五帖）
- **nav**：`anchor-index` 頁側章節錨點索引（IntersectionObserver 高亮當前章節）
- **signature**：手帖跨頁（紙膠帶、手寫鍋記、水彩色票）＋風味測驗滑桿即時生成風味輪並配對果醬
- **illust**：`watercolor-wash` 水彩暈染（純 SVG／CSS，罐身、風味輪、色票暈開皆原創）
- **mood**：`pastel-soft` 米奶油底＋草莓紅／杏黃／葉綠／天藍粉彩

## 色彩
