## 版面原型（帳本欄位）

- **opening**：`catalog-first` — 首頁做成一整面「接觸印樣（contact sheet）」，12 格底片框陳列庫存
- **nav**：`side-rail` — 左側膠捲側欄，含齒孔、片名、幀號（非 topbar）
- **signature**：接觸印樣框 hover 以暗房「顯影」呈現（低反差暗調 → 全反差＋微放大），揭示 EV／機況／價；膠捲齒孔側欄與幀號；比較台雙／三機並排對照結論

- **mood**：`dark-moody` — 近黑畫廊底、象牙墨白、單一暗房安全燈朱紅（僅約 5% 用量）
