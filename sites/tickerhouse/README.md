# TickerHouse — 復古終端機風格示範網站

TickerHouse 是一個虛構的「獨立股票研究台 / 市場資料終端機」示範網站，
以 1980 年代 CRT 磷光綠終端機為視覺語言：黑底、磷光綠主色、琥珀色輔色、
等寬字體、掃描線疊層、打字機動畫與 canvas 繪製的即時跑馬燈與 K 線圖。

> **免責聲明：本站所有股票代號、公司、價格、分析師與研究內容均為虛構的示範資料，
> 不構成任何投資建議。**

## 檔案結構

```
tickerhouse/
├── index.html      首頁：開機自檢動畫、跑馬燈、指數面板（sparkline）、
│                   動態桌面日誌、最新研究列表
├── markets.html    市場頁：120 根日 K 線圖（可切換 CRBX/VLTN/KRNL、
│                   滑鼠十字線讀取 OHLC）、板塊熱度表、漲跌幅排行
├── research.html   研究頁：研究筆記索引、完整範例研究報告（CRBX）、
│                   訂閱表單（假送出、打字機回覆）
├── assets/
│   └── logo.svg    原創 Logo（CRT 螢幕 + 上升柱狀圖 + 等寬字標準字）
├── README.md       本檔
└── SKILL.md        風格技能檔：讓任何 AI 只憑此檔即可重建同樣風格
```

## 使用方式

純靜態網站，無建置流程。直接用瀏覽器開啟 `index.html` 即可；
或在本目錄啟動任一靜態伺服器：

```
python3 -m http.server 8000
# 開啟 http://localhost:8000
```

唯一外部相依：Google Fonts 的 IBM Plex Mono（離線時會退回系統等寬字體，
版面不會壞）。

## 互動速查

| 按鍵 | 功能 |
|------|------|
| `1` / `2` / `3` | 切換 儀表板 / 市場 / 研究 頁 |
| `Q` / `W` / `E` | （市場頁）切換 K 線圖股票 |
| `S` | （研究頁）跳到訂閱區塊 |
| 任意鍵 / 點擊 | 跳過首頁開機動畫（同一分頁工作階段只播一次） |

## 技術重點

- **單檔頁面**：每頁 HTML 內嵌全部 CSS/JS，互相以相對路徑連結。
- **Canvas 圖形**：跑馬燈、sparkline、K 線圖（含成交量、十字線、最新價虛線）
  全部以 canvas 依虛構資料繪製，支援 devicePixelRatio 與視窗縮放。
- **確定性假資料**：K 線由固定種子的 mulberry32 PRNG 生成，
  每次載入畫面一致；其餘價格與表格為硬編碼。
- **CRT 效果**：`repeating-linear-gradient` 掃描線疊層、radial 暗角、
  steps() 閃爍動畫、text-shadow 磷光光暈、方塊閃爍游標。
- **響應式**：窄螢幕收合狀態列、表格橫向捲動、格線降為單欄。
- **無外部圖片 / 無 JS 函式庫**；favicon 為行內 SVG data URI。
