# 光暈九二 HALCYON 92 — 蒸氣波（Vaporwave）

> Design Skills Center 館藏範例站 · 產業 × 風格：**唱片廠牌 × 蒸氣波 Vaporwave**

一間虛構的台北獨立蒸氣波唱片廠牌。整個網站被做成一台名為 **HALCYON OS v9.2** 的混音／卡帶軟體：首頁不是大標開場，而是一個「軟體視窗」（os-metaphor），裡面有母帶播放器、可拖曳的主控台推桿；導覽不在頂端，而是固定在底部、長得像播放器的 **transport bar（bottom-dock）**。整站示範蒸氣波的視覺語彙——CRT 掃描線、透視地平線格點、鉻金屬立體字、日落半圓、洋紅×青×日落橘的霓虹色、半形片假名點綴。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連，共用一致的 bottom-dock 導覽與 footer。
- 所有圖像為原創 inline SVG（卡帶、黑膠、抽象肖像、日落格點）與 CSS，零外部圖片；外部資源僅 Google Fonts（Zen Dots + Space Mono + Noto Sans JP）。
- favicon 為原創 inline SVG data URI；`assets/logo.svg` 為原創鉻金屬字標。
- 互動：可拖曳 HUE 推桿即時 retint（`--hue` HSL 變數）、母帶播放器 transport／播放清單、載體篩選，全部支援 `prefers-reduced-motion` 降級（關閉格點捲動與 VU 跳動）與 RWD（≤520px 不破版）。
- 刻意**不使用跑馬燈**：以「軟體視窗介面＋可拖曳推桿換色＋透視格點日落」作為本站的識別性版面與互動簽名。
- 敏感性：本站所有品牌、人物、作品、價格、地址皆為虛構示意。
