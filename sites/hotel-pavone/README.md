# 孔雀閣 Hôtel Pavone — 大稻埕・裝飾藝術精品旅宿

Design Skills Center 館藏 Vol.02。

**產業 × 風格：** 精品旅宿 × Art Deco 裝飾藝術
**語言：** 繁體中文

## 品牌設定

孔雀閣（Hôtel Pavone）是一間虛構的精品旅宿，座落於台北大稻埕迪化街一段 178 號。前身為茶商一九三一年委託建築師林懋德所築的三層裝飾藝術洋樓，二〇一九年由第三代屋主陳映真主持修復。二十二間客房，附設一樓的「翎羽廳」與頂樓的「翎羽酒吧」。

品牌以「孔雀羽扇」為核心母題：鎏金線條、羽眼（peacock eye）、階梯狀山牆與扇形放射。配色取墨綠與鎏金——爵士年代最典型的奢華對比，佐以孔雀藍作次強調。

## 頁面清單

| 檔案 | 頁面 | 重點 |
|------|------|------|
| `index.html` | 首頁 | 羽扇展開 hero 動畫、品牌沿革、客房編號索引、餐飲雙欄、靜態資訊列 |
| `rooms.html` | 客房 | 四種房型交替左右的「畫盤」版面、進場金框描繪動效、設施格 |
| `dining.html` | 餐飲 | 翎羽廳與翎羽酒吧、含價目的點線引導菜單、對稱羽扇插畫 |
| `reserve.html` | 訂房 | 可驗證的訂房表單（房型 query 預選、確認卡）、住宿須知、街廓地圖 SVG |

## 資產

- `assets/logo.svg`：階梯拱框中的幾何孔雀羽扇 Logo。
- Favicon：inline SVG data URI，簡化羽扇＋墨綠底方框。
- 所有插畫（羽扇、拱窗、屏風、屋瓦、雞尾酒杯、街廓地圖）皆為原創 inline SVG，零外部圖片。
- 外部資源僅 Google Fonts（Noto Serif TC／Poiret One／Cormorant Garamond／Marcellus／Marcellus SC）。

## 風格關鍵字

裝飾藝術、Art Deco、鎏金、墨綠、孔雀羽眼、階梯山牆、扇形放射、對稱與對角、爵士年代、editorial、精品旅宿、大稻埕。

## 動效簽名

首頁的孔雀羽扇於載入時逐片旋轉展開（stagger unfurl），是本站專屬的識別動作；客房頁採進場時金框沿描邊繪出（stroke-dashoffset）。全站無跑馬燈，重點資訊改以靜態資訊列與對稱構成承載。均提供 `prefers-reduced-motion` 降級。
