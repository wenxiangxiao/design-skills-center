# 跳跳格 Zigzag 兒童創造學園 — 孟菲斯 Memphis

> Design Skills Center 館藏範例站 · 產業 × 風格：**兒童教育 × 孟菲斯 Memphis**

一間虛構的台北松山兒童 STEAM 創造學園，服務 4–12 歲的孩子。整站示範 **孟菲斯（Memphis Milano）設計語言**：奶油底＋圓點網格，粗黑描邊、無模糊的硬位移陰影，鮮明的糖果原色（粉／青／黃／珊瑚／葡萄／薄荷），滿版散落的幾何圖形（圓、三角、鋸齒、波浪、方塊）與圓潤字體。動效活潑彈跳——圖形漂浮、卡片 hover 抖動、還有一個可互動的「課程轉盤」。

## 風格關鍵字

孟菲斯、Memphis Milano、Ettore Sottsass、糖果原色、粗黑描邊、硬位移陰影（無模糊）、圓點網格、鋸齒與波浪、散落幾何、terrazzo、圓潤字體（Fredoka）、彈跳動效、玩心。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連。
- 所有圖像為原創 inline SVG／CSS（幾何圖形、課程 icon、轉盤 conic-gradient、老師頭像），零外部圖片；外部資源僅 Google Fonts（Fredoka + Noto Sans TC）。
- favicon 為原創 inline SVG data URI（黃底＋粉圓＋黑鋸齒＋青三角）；`assets/logo.svg` 為原創字標。
- 互動：行動選單、漂浮圖形、課程轉盤（conic-gradient 旋轉＋彈跳結果）、進場揭示、年齡篩選，全部支援 `prefers-reduced-motion` 降級與 RWD（≤560px 不破版）。
- **無跑馬燈**：本站以「漂浮幾何＋課程轉盤＋卡片抖動」承擔動效簽名，刻意不使用捲動橫幅（見 PLAYBOOK §2.1）。
