# FACET 選物 — 選物店 × 極簡留白（css-3d 立體構成）

台北赤峰街的當代選品店示範站。核心概念：每件物件都是一個「六面規格方塊」——品名、材質、尺寸、產地、設計者、定價，一面一則規格。用純 CSS 3D transform 打造可拖曳旋轉的立體構成，把「刻面（facet）」的品牌隱喻變成可操作的介面。

## 品牌設定

- **中文名**：FACET 選物
- **英文名**：FACET
- **產業**：選物店（concept / select shop）
- **風格**：極簡留白（Minimal Whitespace）
- **插畫技法**：css-3d 立體構成（CSS 3D transform 打造的規格方塊與旋轉多面體，零圖片）
- **地點**：台北市中山區赤峰街 12 巷 5 號
