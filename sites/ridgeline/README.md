# 稜線 RIDGELINE — 高山山屋線上預約

> 產業：山屋預約　×　風格：地形測繪風 Topographic Survey
> 全館首個 **Three.js 真 3D 場景**：首頁即一塊可拖曳旋轉／傾斜的立體高山地形。

台灣中央山脈「鏡池山列」（虛構示意）的高山山屋抽籤與訂床服務。品牌主張是「訂床之前，先把山讀懂」——把平面地圖上難以想像的坡度、海拔與山屋位置，攤成一塊可以親手轉一圈的立體浮雕。

## 頁面

1. `index.html` — 概覽。map-first 開場：3D 地形佔據視覺核心，右側路線選擇器連動海拔剖面；沿線山屋卡與「為什麼先看地形」。
2. `reserve.html` — **核心功能**：線上抽籤訂床四步驟（選路線 → 抽梯次 → 選人數 → 聯絡確認），即時試算費用、依餘床上限夾住人數、生成預約編號 `RDG-YYMMDD-NNN` 與床位配置；左側 3D 地形隨所選路線點亮對應山屋。
3. `huts.html` — 山屋檔案。沿稜索引 SVG（點山屋跳檔案）、五座山屋設施表、入山安全須知、山屋管理站團隊。

## 首創說明（登記於 ledger `_first_registry`）

**WebGL / Three.js 真 3D 場景**——本站是館內第一個載入 Three.js（自 cdnjs）並以真 3D 網格為核心體驗的站。程序式高斯山脊 + 分形雜訊生成鏡池山列地形，`toNonIndexed` + `flatShading` 呈現測繪浮雕面感，頂點依海拔上色。符合 PLAYBOOK §2.2 3D 例外條款：3D 為核心元素、非裝飾，並配 `prefers-reduced-motion` 降級與非 WebGL 靜態 SVG 剖面備援。
