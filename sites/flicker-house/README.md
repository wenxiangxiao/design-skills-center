# 明滅戲院 THE FLICKER HOUSE — 走進去坐下的立體放映廳

> 產業：獨立戲院　×　風格：放映廳暗場風 Screening-Room Noir
> 全館首個 **純 CSS 3D 變換空間（perspective + transform3d，零函式庫、零外部圖片）**：首頁是一座可拖曳環視的立體放映廳，選座時畫面即時轉成「你那一席望向銀幕」的視線。

台北大同區保安街一間（虛構示意）單廳修復戲院。它不追首輪、不趕檔期，只把老拷貝擦亮，讓它們在暗場裡再亮一次。品牌主張是「請人入座，而非展示海報」——多數戲院網站是一張平面海報，本站是一座有縱深、有座位、有視線的劇場。

## 頁面

1. `index.html` — 放映廳首頁。**fullbleed-art 開場**：整片主區就是一座純 CSS 立體放映廳（銀幕／光束／側牆／地板／六排絨椅），可拖曳或用 ←/→ 環視；下方是今日四場場次帶、抵達資訊與運作說明。
2. `programme.html` — 本季片單。五部修復拷貝，每部一張以 JS 生成的**原創 halftone 半調網點海報**（各用不同亮度 motif 函式）、導演／年份／片長／格式、劇情簡介與場次，直達選座。
3. `seats.html` — **核心功能**：選片 → 選場次 → 在 `rotateX(33deg)` 斜面立體座位圖點選座席（可選／第 7 排最佳位／已選／已售）→ 右側 sightline 即時把銀幕轉成你那一席的視線與距離讀數 → 逐座選票種算總額 → 姓名／手機驗證 → 生成訂位編號 `MM-YYMMDD-NNN` 與原創 SVG 條碼票根。
4. `about.html` — 戲院與放映師。單廳沿革、選片方式、第 7 排的故事、CSS 3D 自轉膠卷盤、營運資訊表與三位工作人員。

## 首創說明（登記於 ledger `_first_registry`）

**純 CSS 3D 變換空間（transform3d，零函式庫）**——本站是館內第一個「用純 CSS 建立可環視立體空間」的網站。前三個 3D 站都靠 WebGL/Three.js（ridgeline、compositie-optiek）或原生 shader（nordvarsel）；本站刻意反其道，只用 `perspective` + `transform3d` + `preserve-3d` 就搭出一座放映廳，因此輕、快、無依賴、任何靜態主機皆可跑。

真正別站沒有的核心，是把 3D 拿去**服務一個功能**而非裝飾：**選座即見那一席望向銀幕的視線**。你在斜面座位圖上點下座位，右側銀幕會依你選位的列深、偏移即時 `translateZ / scale / rotateY / rotateX`，前排壓境、後排退遠、偏座偏擺，並輸出距離與仰角讀數。拿掉戲院皮膚後剩下的，是一個「以純 CSS 3D 呈現座位視線」的可玩選位器——這是本館其他站都沒有的東西。

符合 PLAYBOOK §2.6 破格條款與 §2.2 3D 例外：3D 為體驗核心非裝飾，且附 `prefers-reduced-motion` 降級與 `@supports not (transform-style:preserve-3d)` 的 2D 平面備援，座位、票價、視線讀數在任何環境都找得到、可操作。
