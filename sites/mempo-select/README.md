> Design Skills Center 館藏範例站 · 產業 × 風格：**選物店 × 孟菲斯 Memphis**

## 風格關鍵字

孟菲斯 Memphis、後現代、飽和撞色、Bacterio 塗鴉底紋、色塊構成（flat-shape）、粗黑 3px 描邊、硬邊位移陰影、圓點方塊三角、歪斜直角、catalog-first、bottom-dock、幾何碎片彈射、游標噴濺拖尾。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連，共用一致的底部欄導覽與 footer。
- 所有圖像為原創 inline SVG／CSS／canvas（幾何選品插畫、Bacterio 底紋、位置示意圖、游標粒子），零外部圖片；外部資源僅 Google Fonts（Archivo Black + Space Grotesk + DM Mono）。
- favicon 為原創 inline SVG data URI；`assets/logo.svg` 為原創字標。
- 互動：分類篩選、進場揭示、加入選物袋計數＋幾何碎片彈射、游標幾何噴濺拖尾（canvas），全支援 `prefers-reduced-motion` 降級與 RWD（≤560px 不破版）。本站**不使用跑馬燈**——動效簽名是「幾何碎片彈射＋游標噴濺拖尾＋選物袋翻動計數」。
