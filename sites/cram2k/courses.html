>> ==== [ A 區 · 學測衝刺班 ] ============================================ ◆

DISC 01

學測總複習班 學測

國英數自社五科整合，從高一觀念補到考前總複習。每週一份仿真模考、專屬導師追進度，適合想一次搞定五科的高三生。

時段週一～週五 19:00–21:30
名師五科名師群
費用$28,000 / 期

DISC 02

國文名師班 學測

王硯之老師「古文全解」，把三十篇核心古文拆成故事講。白話閱讀策略＋文意題型解碼，讀懂比背多分更重要。

時段週二 19:00–21:30
名師王硯之
費用$13,200 / 期

DISC 03

英文魔王單字班 學測

陳語柔老師帶你 16 週吃掉學測 7000 字。字根字首拆解＋每日雲端小測驗，閱讀測驗與翻譯寫作同步練。

時段週三 19:00–21:30
名師陳語柔
費用$13,200 / 期

DISC 04

數學 A 衝刺班 學測

李國棟老師「題型破解」，把學測數 A 拆成 40 種必考題型。先講觀念再狂刷題，錯題本回收檢討到懂為止。

時段週四 19:00–21:30
名師李國棟
費用$14,400 / 期

DISC 09

國寫作文班 學測

王硯之老師專攻國寫兩大題：知性題的論述結構、情意題的畫面經營。八週八篇批改，一篇一篇把分數磨上去。

時段週六 09:30–12:00
名師王硯之
費用$9,600 / 期

>> ==== [ B 區 · 分科測驗班 ] ============================================ ◆

DISC 05

物理分科班 分科

張哲宇老師「手寫詳解」，力學電磁近代物理全範圍。每題附完整推導手稿，觀念題與計算題兩手都要硬。

時段週一 19:00–21:30
名師張哲宇
費用$15,600 / 期

DISC 06

化學分科班 分科

林思妤老師「實驗題本」，把有機無機與計算章節整成表格記憶卡。歷屆實驗設計題逐年拆解，考點一個都不漏。

時段週三 19:00–21:30
名師林思妤
費用$15,600 / 期

DISC 07

生物分科班 分科

周書瑋老師「圖解記憶」，細胞遺傳到生態全用手繪概念圖串起來。醫農牛校系必修，實驗閱讀題特訓加強。

時段週五 19:00–21:30
名師周書瑋
費用$15,600 / 期

DISC 08

社會總複習 分科

歷史地理公民三合一，時間軸、地圖與時事議題整合教學。跨科混合題型演練，社會組衝頂標的最後一塊拼圖。

時段週六 14:00–17:30
名師史地公名師群
費用$16,800 / 期

>> ==== [ C 區 · 國中會考 / 先修 / 自習 ] ================================ ◆

DISC 10

國中會考班 會考

國英數三科同步，會考題型與寫作測驗一起練。分級小班、每月定期模考，目標三科五標、序位往前衝。

時段週六 09:00–12:00
名師國英數名師群
費用$12,000 / 期

DISC 11

高一先修班 先修

暑假先把高一數學與英文文法打底，開學不卡關。輕量進度、觀念優先，讓國三升高一的銜接坡道緩一點。

時段暑期 週二四 14:00–16:30
名師先修師資群
費用$8,800 / 梯

DISC 12

K 書自習室 自習

獨立桌燈、個人插座、恆溫冷氣與飲水機。月租制，早上十點開到晚上十點，補習班學員享折扣，安靜是唯一的規則。

時段每日 10:00–22:00
座位60 席獨立桌
費用$1,800 / 月
