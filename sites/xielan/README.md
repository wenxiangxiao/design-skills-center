# 謝籃 XIELAN 廟埕選物

**產業**：廟宇文創　·　**風格**：拓印風 Rubbing（碑拓／朱印）

一間虛構的台南廟埕文創選物店，開在普濟殿廟埕旁。設計語言取自傳統碑拓——白線黑底、朱紅印章、金墨回紋。首頁不是招牌大標，而是一頁「目次」，把求籤、御守、拓印文具、關於廟埕四件事排成廟口告示般的索引；核心功能是把廟裡的求籤搬上螢幕：搖籤筒、抽出籤支、開籤揭示籤詩與吉凶，開籤同時附一枚稀有度不定的御守盲盒。嚴格朱砂紅×墨黑雙色域，米紙提亮、金線點綴。

## 八維配方

> **廟宇文創 × 拓印風 Rubbing × 當代 × 台灣本土 × 溫柔療癒 × 紙感（拓印墨拓）× 中等密度 × 抽獎/扭蛋/盲盒**

- **era**：當代——線上求籤、盲盒玩法，語彙卻承接廟埕碑拓的老傳統。
- **locale**：台灣本土——台南普濟殿、廟埕、謝籃、擲筊抽籤的在地信仰場景。
- **voice**：溫柔療癒——「先靜心，再搖籤」「誠心不誠心，籤知道」，撫慰而不喧嘩。
- **texture**：紙感——碑拓墨拓的白線黑底、宣紙顆粒、朱印蓋壓。
- **density**：中等——目次與圖鑑清爽，但籤詩、御守、機率資訊完整。
- **function（核心可玩功能）**：**抽獎/扭蛋/盲盒**——搖籤筒（籤筒晃動）→ 籤支彈出顯示籤號 → 開籤，籤詩以碑拓「上墨」clip-path 由上而下拓現、逐句浮字、朱印落款蓋章；同時滾動御守盲盒稀有度（常 60%／稀 30%／極稀 10%），揭示一枚御守；可「再求一支」並累計本次求籤次數。
