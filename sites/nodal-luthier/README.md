## 首創（_first_registry 認領項）

**模態聲學模擬作為可操作的調音台。**

全館既有的物理／空間類首創中，使用者操作的都是「看得見的東西本身」——熔融玻璃的輪廓、戲偶的手、岩牆上的岩點、船的舵。本站是第一個讓使用者操作**看不見的參數**（厚度分佈），再由一個特徵值問題把結果**翻譯成看得見的圖形**（砂粒排出的節線）回答你。

技術上包含三件全館沒有出現過的事：

1. **駐波節線場模擬**——三個假設模態形狀寫在小提琴面板輪廓內，砂粒依「振幅越大跳得越亂、節線上不跳」的規則隨機遊走並被節線捕獲，圖形是模擬出來的，不是預先畫好的路徑。
2. **瑞利商即時求解**——面板頻率由 `√(Σ 彎曲應變能·t³ ÷ Σ 質量·t)` 在 48×92 網格上每次刨削後重算，因此厚度分佈與頻率是真的耦合，不是查表。
3. **有號敏感度場**——對每一格算 `∂f/∂t`，因為剛性項隨 t³、質量項隨 t，節線附近去料會讓頻率**上升**。這個正負分佈直接畫成雙色場，是使用者做決策的依據。
