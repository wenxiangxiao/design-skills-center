# 潮汐訊號 TIDE SIGNAL — 基隆港聲音地景計畫

> **全館第一個有聲音的網站。** 不使用任何音訊檔案——雨、浪、霧笛、廟口低語、船機引擎，
> 五個聲道全部以原生 Web Audio API 即時合成（噪音源＋濾波器＋LFO＋振盪器）。
> 聲音在使用者點擊「接上訊號」後才啟動（符合瀏覽器 autoplay 政策），
> 提供總音量滑桿、靜音鈕與「停止所有聲音」，預設中低音量。

虛構的聲音藝術團隊「潮汐訊號」，在基隆港做田野錄音與聲景保存。
本站把整座港畫成一塊 PCB 電路板：五個聲道模組以銅箔走線連到中央混音 IC，
開啟聲道時走線亮起、訊號脈衝沿線流動——**聽得到的，同時看得到。**

## 配方代碼

當前聲景可編成一段代碼，例如 `TS-R7050W4050F0040T0050E0045`：
`TS-` 之後每四碼一段——聲道字母（R 雨／W 浪／F 霧笛／T 廟口／E 引擎）＋兩位音量＋兩位參數。
音量 `00` 代表該聲道關閉。頁面提供輸入框，貼入代碼即可重現該聲景；亦相容兩位數簡碼（如 `TS-R70W40`）。

## 聲音合成摘要（無音訊檔）

- **雨**＝白噪音帶通嘶聲＋隨機排程的短促噪音脈衝（密度可調）
- **浪**＝噪音→低通→雙 LFO 調變振幅與濾波頻率（週期可調）
- **霧笛**＝雙鋸齒＋次八度正弦→低通→定時包絡鳴放（間隔可調）
- **廟口低語**＝雙帶通噪音人聲游走＋偶發正弦鐘聲（密度可調）
- **船機引擎**＝低頻鋸齒＋律動 AM＋低通轟鳴噪音（轉速可調）

完整節點鏈見 `SKILL.md` 第七章。

## 降級與無障礙

`prefers-reduced-motion` 時：走線脈衝停止移動、波形以低頻率更新、選單改為無動畫切換；
「停止所有聲音」常駐於音訊塢。全站 RWD 至 360px，≤560px 時配置器改直向堆疊。
