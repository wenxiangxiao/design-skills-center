## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連，共用一致的側欄導覽與 footer。
- 所有圖像為原創 inline SVG／CSS（交叉排線本草藥材、脈息呼吸圓環、位置示意圖、紙張顆粒 feTurbulence），零外部圖片；外部資源僅 Google Fonts（Noto Serif TC + Noto Sans TC + DM Mono）。
- favicon 為原創 inline SVG data URI（青囊＋草木線刻）；`assets/logo.svg` 為原創字標＋交叉排線徽記。
- 互動：側欄導覽、本草索引停序逐味揭示、脈息呼吸圓環，全支援 `prefers-reduced-motion` 降級與 RWD（≤560px 側欄轉為橫向、不破版）。本站**不使用跑馬燈**——動效簽名是「本草索引停序逐味揭示＋脈息呼吸圓環＋交叉排線本草描繪」。
- 醫療敏感：全站標示虛構示意與「不構成醫療建議」免責。
