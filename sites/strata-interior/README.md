## 開場與導覽原型

- **開場（opening）**：`chapter-scroll` — 無大標 hero，首頁以編號章節逐節捲動展開，開場即一組雜誌封面對開。
- **導覽（nav）**：`topbar` — 置頂列，上方附 running head（刊物名／期號）與細線，編輯刊物感。
- **動效簽名（signature）**：跨頁對開大圖說＋SVG 平面線稿隨捲動 stroke-dashoffset 一筆描繪浮現＋章節頁碼角標（folio）。

## 技術

- 每頁單檔 HTML，CSS/JS 全 inline，頁面間相對路徑互連。
- 原創 `assets/logo.svg` 與 inline SVG data URI favicon；所有插圖／平面圖／肖像為原創 SVG，零外部圖片。
- 外部資源僅 Google Fonts（Fraunces／Archivo／Noto Serif TC／Noto Sans TC）。
- 全站支援 `prefers-reduced-motion`（描繪動畫降級為直接顯示完成線稿）與 RWD（≤820px 對開改為單欄，隱藏中縫）。
