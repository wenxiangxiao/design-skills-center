# 日安到宅居家護理所（Goodday Home Nursing）

台北「日安到宅居家護理所」品牌示範網站——以**醫院／捷運導視系統（Wayfinding）語言**整站化的居家護理所範例站。三條到宅護理服務線（W 傷口線、T 管路線、C 慢病線）比照捷運路線編碼與配色，首頁開場即為互動式「大台北服務路網圖」。

> 本站內容為虛構示意，非醫療建議。所有機構、人名、立案字號、收費與合作單位均為設計展示用途。

## 八維配方

| 維度 | 設定 |
|---|---|
| 產業 industry | 居家護理所（衛福部立案・到宅護理服務） |
| 風格 style | 導視系統 Wayfinding（醫院／捷運指標語言整站化） |
| 年代 era | 當代 |
| 場域 locale | 台灣本土（大台北行政區、台灣收費與健保語境） |

| 質感 texture | 標示牌材質感：霧面金屬漸層看板＋反光膜色帶（斜向高光） |
| 密度 density | 中等 |
| 功能 function | 到宅訪視預約系統（三步流程、週曆名額、預約代號） |

## 硬性輪替約束（PLAYBOOK）

- **開場原型 map-first**：首頁第一屏即「服務路網圖」——大台北服務區畫成捷運式路線圖 SVG，三線交會、站點＝行政區，點站點顯示該區駐區護理師與班表摘要。
- **導覽原型 bottom-dock**：主導覽固定於視窗底部，做成月台指標列（含前往方向箭頭、當前頁黃色指示帶）。
- **插畫技法 flat-shape pictogram**：全站圖標為無描邊色塊自繪 SVG（屋形、管路、足部、日曆、警示三角等）。
- **mood**：中間調淺灰藍底（#B9C4C9 系），非米白紙感、非深黑底。
- **功能**：預約三步流程完整可玩——選服務線與項目 → 週曆選時段（顯示剩餘名額、額滿變灰、週日休診、週六僅上午）→ 前端驗證表單 → 產生預約代號與站牌票卡式確認畫面。

## 設計要點

- **三線色票**：W 傷口線 `#BF4522`（暖紅）、T 管路線 `#1F6FA5`（號誌藍）、C 慢病線 `#2E7D4F`（標示綠）；警示黃 `#F0B429` 作為方向與強調色；板岩深色 `#22343B` 作為看板底。
- **字體**：Noto Sans TC（400/500/700/900）＋ Barlow Condensed（DIN 感西文，用於編號、代碼、英文標示）。
- **每頁單檔 HTML**（CSS/JS inline）、相對路徑互連、一致 header／footer／bottom-dock；favicon 為原創 inline SVG data URI；零外部圖片，僅載入 Google Fonts。
- **無障礙**：站點與時段皆可鍵盤操作、`aria-pressed`／`aria-current`／`aria-live` 狀態、`prefers-reduced-motion` 降級、RWD 至 560px 以下不破版。
