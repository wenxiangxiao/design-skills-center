## 首創（登記於 ledger `_first_registry`）

連續氣球的扭結拓樸建構作為互動範式：一根有限長度的模型氣球，以一連串扭轉（基本扭／鎖扭／捏扭）被分節、相鎖成可辨識造型；**長度是消耗性資源、過緊會爆、末端必須留軟尾**。與 gluhwerk-glass（旋轉對稱剖面的熔融玻璃物理配置器）、muli-arbor／sporekey（量測／檢索的決策）、shaft-eight（織法矩陣渲染平面布料）、foldpoint（剛性摺疊運動學）皆不同：本站的互動對象是一條一維充氣介質的**離散扭結拓樸**與其資源約束（充氣長度）與失效模式（爆破／缺料），造型即扭法序列的直接結果。
