一個以「先辨識、再談食用」為唯一守則的野外真菌同好社。整站不是圖鑑查詢，而是一把**真正的二分檢索表**：你手上有一朵菇，鑑定台每次只問一個性狀、逼你二選一，走到物種為止；看不清楚的岔口，本台**拒絕替你猜**，而通往可食種的最後一格，一定是在替你排除某個致命的相似種。

## 頁面

1. **index.html — 野菇鑑定台**：檢索岔口開場。首屏即檢索表第一個岔口（孢子印色），內容沿你走的分支往下長。兩種模式：逐岔口走檢索、或直接輸入觀察到的性狀讓本台自動判讀。走到白孢有環的「霜柄白傘 vs 白毒傘」時，基部未確認就把判定退回「不可食」。
2. **atlas.html — 蕈譜**：24 種常見有柄傘菌，每格插圖為該種的孢子投影；可依食性／孢子印色／生態多條件即時篩選＋關鍵字＋計數。
3. **method.html — 檢索法與安全守則**：二分檢索表全文、性狀觀察術語、採前六戒。完整靜態可讀，不需 JavaScript。
4. **foray.html — 採集會報名**：三步驟表單（選場次→填資料含驗證→取得留位號 SK-YYMMDD-NNN）。
