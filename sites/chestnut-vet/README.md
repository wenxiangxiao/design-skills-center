# 栗子動物醫院 — 兒童繪本風（Children's Picture Book）

> Design Skills Center 館藏範例站 · 產業 × 風格：**動物醫院 × 兒童繪本風**

一間虛構的社區型動物醫院，開在虛構的「栗子鎮」中央公園旁。整站示範 **兒童繪本風（Children's Picture Book）**：手繪蠟筆線稿、水彩暖色、圓潤字體、故事書般的版面與敘事口吻，把「看醫生」這件讓毛孩緊張的事，說得像翻一本溫柔的繪本。

## 版面與骨架（對照 PLAYBOOK §2.4）

- **開場原型 opening：`map-first` 地圖開場**——首頁一進來就是一張手繪的栗子鎮地圖，沿著小溪、經過小溪橋、看到紅屋頂就是診所。地圖上的站點（車站、公園、小溪橋、診所、散步的動物）可互動 hover 顯示標籤，診所大頭針持續彈跳。地圖本身就是 hero，而非置中大標。
- **導覽原型 nav：`hamburger-full` 全螢幕選單**——頂列只放 Logo 與一顆蠟筆漢堡鈕，點開是整頁「翻開繪本 · 選一頁」的巨字故事書選單。（本次為連續 4 站中的非 topbar 導覽。）
- **動效簽名 signature：水彩蠟筆手繪動物 ＋ 翻頁書角互動**——每頁右下角有一枚折起的書頁角，hover 時書角捲大、寫著「翻頁看服務 →」，點擊翻到下一頁；搭配落葉飄落與大頭針彈跳。全館唯一。

## 頁面

1. `index.html` — 地圖開場首頁：手繪栗子鎮地圖、診所介紹、看診時間告示牌、服務預覽、書頁角。
2. `services.html` — 診療服務：六項服務的手繪動物插畫卡與參考費用、諮詢引導。
3. `visit.html` — 醫師與預約：兩位醫師肖像、繪本紙條式預約表單（示範用，不真送出）、車站到診所的路線圖。

## 去AI化重點

- 無置中大標模板；首頁是可互動的手繪地圖。
- 圖像全為原創手繪 SVG（地圖、動物、醫師肖像、路線圖），零外部圖片；外部資源僅 Google Fonts。
- 導覽採全螢幕繪本選單，非制式置頂列，補足全館 nav 多樣性。
- 無跑馬燈——繪本風不需要捲動橫幅，動效重點交給書頁角與地圖互動。
- 敏感資訊（費用）標示為參考、皆為虛構示意。
