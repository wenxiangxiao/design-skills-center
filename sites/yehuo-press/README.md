## 頁面

1. `index.html` — 首頁。以**滿版木刻展覽海報（fullbleed-art）**開場，載入時海報由中心 clip-path「壓印」揭開；現展介紹、作品牆（滑鼠移過套色錯位）、版種、檔期預告、參觀資訊。
2. `exhibitions.html` — 檔期總表。現正展出／即將登場／過往展覽以粗黑表格列出，每筆附木刻縮圖、會期、藝術家。
3. `workshop.html` — 版畫工作坊。四種版種介紹（木口木版／凸版／絹印／蝕刻）＋秋季課程時程表＋報名流程與資訊。

## 版面原型

- **opening**：fullbleed-art（首頁以邊到邊的原創木刻展覽海報為 hero，非置中大標＋兩顆按鈕、無漸層）
