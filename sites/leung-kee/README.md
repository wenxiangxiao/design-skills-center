## 首創說明（登記於 ledger `_first_registry`）

**真實時刻晝夜引擎（wall-clock day/night）**——本站係全館第一個「用訪客真實本地時鐘即時驅動晝夜」嘅網站。前面幾個站分別做過有聲（tide-signal）、生成式版面（dice-needle）、Three.js 3D（ridgeline／compositie-optiek）、原生 shader 極光（nordvarsel）、鍵盤駕駛（first-gear）、純 CSS 3D（flicker-house）、canvas 物理（gluhwerk-glass）。本站唔碰 3D,改由「時間」本身做媒介：一條九個錨點嘅時刻曲線,線性內插出天色漸層（`--sky-a/--sky-b`）、霓虹 bloom 強度（`--on`）、環境暗度（`--dark`）同落閘狀態,每次載入讀 `new Date()`,再配一條可預覽任一時刻嘅桿。
