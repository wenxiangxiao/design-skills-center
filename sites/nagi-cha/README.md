# 凪 NAGI 日本茶 — 和風極簡（Japanese Minimalism / Wafū）

> Design Skills Center 館藏範例站 · 產業 × 風格：**茶葉品牌 × 和風極簡**

一間虛構的台北日本茶專門店。以「一服の静けさ（一杯茶的靜）」為主軸，整站示範**和風極簡（和風・ミニマル）**：大量留白（間・ma）、直書標題、明朝體與黑體的疏密對比、米白墨黑苔綠的自然色調、一筆圓相（円相）動畫，以及不對稱的編輯式版面。

## 品牌設定

- **中文名**：凪 NAGI 日本茶
- **英文名**：NAGI — Single-Origin Japanese Tea
- **地點**：台北市大安區永康街 7 巷 3 號（虛構）
- **一句話**：凪，是海面歸於平靜的那一刻——把季節與土地，端成一杯茶。
- **人物**：主理人 森田凪、茶師 陳語彤、焙茶職人 吉田誠（皆虛構）
- **代表色**：紙白 `#F4F1EA`／墨黑 `#1C1B18`／苔綠 `#6E7355`／朱 `#B24A38`／石灰 `#C7C1B4`

## 風格關鍵字

和風極簡、間（ma）、侘寂、円相、直書（tategaki）、明朝體、Shippori Mincho、疏密對比、自然色調、細硬分隔線、不對稱編輯排版、緩慢揭示、無圓角、零陰影。

## 技術

- 每頁單檔 HTML，CSS／JS 全 inline，頁面以相對路徑互連，共用一致的導覽列與 footer。
- 所有圖像為原創 inline SVG（円相、茶葉、急須、茶碗、冷萃壺、遠山霧線），零外部圖片；外部資源僅 Google Fonts（Shippori Mincho + Noto Serif TC + Zen Kaku Gothic New）。
- favicon 為原創 inline SVG data URI（円相＋苔綠點）；`assets/logo.svg` 為原創字標。
- 互動：行動選單、円相描繪、進場揭示、互動茶時計倒數，全部支援 `prefers-reduced-motion` 降級與 RWD（≤560px 不破版）。本站不使用跑馬燈——動效簽名是「一筆円相」與「茶時計」。
