## 頁面

1. `index.html` — 觀測台。**os-metaphor 開場**：整個首頁是一座極光主控台，中央即時 WebGL 光幕＋五格即時儀表（Kp／太陽風／Bz／雲量／可見度）＋可拖曳的 Kp 滑桿即時改變天空；下接三把量測、各觀測點今晚分數、Kp 級距對照。
2. `varsel.html` — **核心功能**：極光預報查詢。選觀測點＋選夜晚 → 排出 18:00 至翌 06:00 的逐小時可見度時間軸（seeded PRNG，同條件恆同結果）、標出今夜最佳窗口與建議；**點任一時段，右上天空重繪為那一小時的 Kp 光幕**。函數與生成藝術在此融為一體。
3. `himmel.html` — 極光指南。成因（太陽風→磁力線→氧氮發光，附原創 SVG 注入示意圖）、Kp 指數判讀表、現場觀測清單、手機／相機拍攝設定。
4. `stasjon.html` — 觀測站。原創 SVG 峽灣觀測網地圖、守夜團隊、開放時間、方案（皆標示虛構）。

## 首創技術（登記於 ledger `_first_registry`）

**原生 WebGL 片段著色器即時極光（無函式庫，Kp 驅動）。** 一個全螢幕四邊形，片段著色器以 fbm 雜訊生成三層舞動的極光簾幕與星空；uniform `uKp`（0–9）即時決定強度、簾幕垂降高度與色相漸變（低 Kp 綠青、高 Kp 疊上紫紅洋紅）。預報頁把每一小時算出的 Kp 餵進同一著色器，於是「查預報」與「看天空」是同一件事。降級：`prefers-reduced-motion` 凍結為靜態幀；無 WebGL 環境自動切換為原創 SVG 漸層極光帶備援（零外部圖片）。
