# 明算聯合會計師事務所 MINGSUAN — 瑞士國際主義（Swiss International）

> Design Skills Center 館藏範例站 · 產業 × 風格：**會計師事務所 × 瑞士國際主義 Swiss International**

一間虛構的台北信義區聯合會計師事務所。**明算**把首頁做成一份「可用的試算表工作表」——不是大標 hero，而是帶有公式列、儲存格參照（A1、C4）、欄列標頭與分頁標籤的真實表格介面。事務所的名稱、標語、服務、數字，全部就是表格裡的儲存格；選取框會在儲存格之間移動，公式列同步顯示每一格背後的「公式」。這一頁本身就是產業敘事：會計師的世界，就是一張又一張算得清清楚楚的工作表。

整站示範 **瑞士國際主義（Swiss International / International Typographic Style）**：嚴謹網格、齊左的 Archivo 無襯線黑體、2px 實線分格、零圓角、以資訊層級而非裝飾說話。為與館內既有的瑞士風站（RASTER 美術館—紅、CIVICA 開放資料—國際藍、MERIDIAN 地產—紅）拉開，本站改用**會計專屬的「借貸雙色」訊號系統**：正值螢綠 `#1A7F5A`、負值帳紅 `#C0392B`、選取藍 `#1B5FD0`，並全程以 IBM Plex Mono 呈現數字與公式，強化「試算表」的體感。

## 版面與骨架（對照 PLAYBOOK §2.4）

- **開場原型 opening：`os-metaphor` 軟體介面隱喻**——首頁主體是一套試算表工作表軟體：頂部固定 `公式列`（儲存格參照框 + fx + 公式內容 + 就緒狀態），下方是欄標頭（A–F）、列號與內容儲存格，底部是分頁標籤（總覽／服務項目／事務所）。介面隱喻直接服務產業敘事——會計事務所＝工作表。
- **導覽原型 nav：`side-rail` 垂直側欄**——左側固定 236px 側欄，含 2px 格線 logo、編號式垂直選單、地址電話與「內容皆為虛構」聲明；手機收合為頂部橫列。非 topbar，滿足連續 4 站至少 1 站非 topbar 的規則。
- **動效簽名 signature：試算表選取框在儲存格間游走＋公式列同步顯示該格公式＋=SUM/AVERAGE 數字逐格加總跑動＋分頁標籤切換工作表**。選取框以 `inset box-shadow` 呈現不位移版面，可自動巡格也可點擊選取；數字以 `IntersectionObserver` 觸發、`requestAnimationFrame` 緩動累加；全館唯一的「可操作試算表」互動。

## 頁面

1. `index.html` — 總覽工作表：公式列＋工作表首頁（事務所抬頭、標語、三大服務、借貸雙色數字、諮詢 CTA、分頁標籤），下接「為什麼是明算」編號索引與「數字一覽」借貸雙色統計。
2. `services.html` — 服務項目工作表：12 項服務的「項目／內容／週期／費用起」試算表式清單（4 大類），與「從第一次見面到每年結案」四步合作流程。
3. `about.html` — 事務所 · 聯絡：事務所理念、兩位主持會計師資料卡（原創 SVG 頭像）、2009–2024 沿革索引，以及含地址電話與示意預約表單的 `#contact` 聯絡區。

## 去AI化重點

- 無紫藍漸層、無置中大標＋兩顆按鈕＋三張圓角卡片模板；全站零圓角、以 2px 實線網格與試算表儲存格為骨架。
- icon 與頭像皆為原創 inline SVG（格線 logo、儲存格頭像），favicon 為原創 inline SVG data URI；零外部圖片，外部資源僅 Google Fonts（Archivo／Noto Sans TC／IBM Plex Mono）。
- 文案為具體、可信的虛構內容（字號、地址、電話、費用區間、人物口頭禪），無 Lorem ipsum、無 AI 腔套語。
- 未使用跑馬燈；動效簽名由「試算表選取＋公式列＋數字加總」承擔，屬全館未見的獨特互動。
- 敏感題材（財稅金額）全數標示「虛構示意」，並於頁尾聲明不構成任何會計、稅務或法律意見。
