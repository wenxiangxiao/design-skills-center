# 恆準鐘錶維修所 HENGJUN Horological Atelier

**產業**：鐘錶維修　·　**風格**：解剖圖鑑／博物學（Anatomical Atlas）

一間虛構的台北機械鐘錶維修所。不賣新錶，只做解體保養與維修。首頁不是招牌大標，而是一台「維修進度查詢台」——輸入收件時的工單號，就能看見你的錶此刻走到解體、洗油、組裝或校時的哪一格；右側那張博物學風格的等角爆炸機芯圖，會隨進度一枚一枚把齒輪裝回去。整站以炭黑夾板為底、黃銅髮線與朱紅寶石為記，像一本被翻舊的鐘錶解剖圖鑑。
