**產業**：咖啡廳（自家烘焙）　**風格**：復古印刷 Risograph　**語言**：中英混

一間虛構的台南正興街自家烘焙咖啡館。店名取自孔版印刷術語「疊印」（overprint）——兩層油墨交疊會印出第三種顏色，正如豆、水與沖煮者疊出杯中的味道。全站以孔版印刷的語彙鋪陳：暖紙米白 `#F3ECDD`、油墨黑 `#22201C`，加上三支螢光油墨——螢光粉 `#FF4D8D`、湖水藍綠 `#1F9E8F`、螢光橘 `#FF7A2F`，交界處以 `mix-blend-mode:multiply` 疊出第三色，並鋪一層極淡的半調網點顆粒。

刻意避開「大標 hero」——首頁是一頁**無 hero 的章節捲軸**（chapter-scroll），把咖啡館的故事分成序／選豆／手沖／空間／到訪五章；導覽是左側固定的**章節錨點索引**（anchor-index），隨捲動高亮當前章。招牌互動在第 3 章「手沖」：右側的濾杯會隨你捲動注水、粉層 bloom 綻放、下壺咖啡一點一點滿起來（scroll-driven 沖煮進度），四個步驟同步點亮；各章標題以油墨滾筒橫掃「印刷」出來（roller wipe）。

## 開場與導覽原型

- **開場（opening）**：`chapter-scroll` — 首頁沒有大標 hero，而是一條把品牌故事分成五章的捲軸，往下捲＝往下讀。
- **導覽（nav）**：`anchor-index` — 左側固定垂直章節索引（01–05），隨捲動高亮當前章；內頁的章節項改連回 `index.html#錨點`，並以底部兩枚頁鈕連到菜單與手沖課（≤900px 收合為漢堡開關的抽屜）。
- **動效簽名（signature）**：第 3 章手沖濾杯的**水位與下壺咖啡量隨捲動注入**（scroll-driven 沖煮進度）＋粉層 bloom 綻放漣漪＋章節標題以油墨滾筒橫掃「印刷」揭示（roller wipe）＋半調網點疊印質感。與同館的「疊瓣花社 Petal Press（花店×Risograph）」不同：後者以色版開關與套準失調 hover 為主，本站的核心是「捲動即沖煮」的敘事互動。

## 技術

- 每頁單檔 HTML，CSS/JS 全 inline，頁面間相對路徑互連，全站共用左側 anchor-index 導覽與 footer。
- 原創 `assets/logo.svg`（兩枚錯位疊印圓環＋疊印字標）與 inline SVG data URI favicon（粉／綠疊印圓）；濾杯、下壺、老屋、豆票、步驟圖示皆為原創 SVG，疊色用 `mix-blend-mode:multiply`，零外部圖片。

- 全站支援 `prefers-reduced-motion`（沖煮進度直接顯示完成、關閉滾筒揭示與滴水動畫）與 RWD（≤900px 左欄收為抽屜、手沖改上下堆疊、豆卡與格線轉單欄）。
