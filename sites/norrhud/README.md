# Norrhud 諾赫・北方肌膚

北歐簡約（Nordic Minimalism）風格的保養品牌示範站。骨白留白、髮絲線分隔、克制的植物綠與暖中性、Fraunces 襯線與 Jost 幾何無襯線，安靜的進場淡入。刻意做成「深色工業風」的相反——輕、透、慢。

## 品牌設定

- **品牌**：Norrhud（諾赫，norr＝北方，hud＝肌膚）
- **產業**：保養品牌（潔顏／化妝湯／精華／面霜／修護膜／護唇）
- **故事**：創辦人周思頡於挪威特羅姆瑟客居的冬天，領悟「肌膚在極端氣候需要的其實很少，但每件都得對」。二〇一九年於台北大安永康街老公寓一樓成立概念店，小批次製作。
- **成分**：雲杉、沙棘、樺樹汁、泥炭、乳木果、葵花、荷荷芭、菸鹼醯胺——寒地植物為主，可溯源。
- **概念店**：台北市大安區永康街 18 號 · 02-2708-4196 · 週三–週日 12:00–20:00
